# 第二单元 牙龈疾病

| 疾病                 | 病因                 | 诊断要点                                             |
| :----------------- | :----------------- | :----------------------------------------------- |
| 慢性龈炎（单纯性龈炎）（边缘性龈炎） | 放线菌                | 下颌前牙区最重、探诊后出血；色、形、质、点彩消失                         |
| 青春期龈炎              | 牙菌斑是始动因子；性激素加强炎症反应 | 12 ~ 18 岁青少年，前牙唇侧牙龈乳头的球状突起                       |
| 妊娠期龈炎              | 中间普氏菌；性激素加强炎症反应    | 妊娠期女性，前牙区牙龈鲜红、高度水肿、肥大、瘤样突起、8 个月达高峰、妊娠 4 ~ 6 个月切除 |
| 白血病的龈病损            | 血涂片检查；幼稚血细胞浸润      | 出血不易止、牙龈肿大发绀、苍白，可有坏死、溃疡、低热、乏力，切忌活检手术，急性期不洁治      |

青春期龈炎、妊娠期龈炎、白血病的龈病损的菌斑量、牙石较少；探诊易出血

| 疾病               | 病因             | 诊断要点                                                      |
| :--------------- | :------------- | :-------------------------------------------------------- |
| 药物性牙龈增生          | 苯妥英钠、硝苯地平、环孢素  | 服药史，龈乳头实质性增生，球状突起，增生不超过牙冠面积的 2/3；上颌前牙容易移位                 |
| 牙龈纤维瘤病           | 不明确，可有家族史      | 乳牙、恒牙均可发生；累及全口牙龈缘、龈乳头和附着龈，增生牙龈颜色正常（上颌磨牙腭侧最严重）             |
| 急性坏死性溃疡性龈炎（ANUG） | 梭形杆菌、螺旋体、中间普氏菌 | 起病急，病程短，龈乳头坏死虫蚀状、反波浪线状、疼痛明显，自发性出血，腐败性口臭，下前牙区最重，不波及附着龈走马牙疳 |
| 急性龈乳头炎           | 机械或化学刺激        | 局部牙龈乳头发红肿胀、探诊和吸吮时易出血，有自发性的胀痛和明显的探触痛                       |

药物性牙龈增生、牙龈纤维瘤病的牙龈增生、质地变韧、不易出血；
