# 第六单元 口腔颌面部感染

## 第一节 概论

#### （一）感染（`infection`）

**定义**：指由各种**生物性因子**在**宿主**体内繁殖及侵袭,导致**机体**产生以**防御**为主的一系列**全身**及**局部**组织反应的疾患。
生物性因子 + 宿主 → 机体防御→全身 + 局部的组织反应。

#### （二）口腔颌面部感染的特点（易扩散）
①间隙多②静脉多③淋巴多
1. 颜面部血液循环丰富，鼻唇部**静脉**又常**无瓣膜**，致使在**鼻根至两侧口角**区域内发生的感染易向**颅内**扩散，而被称为面部的 **“危险三角区”**
2. 口腔颌面部正常时即有大量微生物存在，当遭受损伤、手术或全身抵抗力下降时会导致正常微生物生态失调的==内源性或外源性感染==发生。
3. 颜面及颌骨周围存在较多的==潜在性颈膜间隙==。其间含疏松的结缔组织（又称==蜂窝组织==），形成感染==易于蔓延==的通道。
4. 面颈部感染可以通过颈深筋膜中层沿气管前间隙、内脏血管间隙和内脏血管后隙向颈部和纵隔扩散，形成更为==广泛和严重的颈部及纵隔脓肿==。
5. 随着广谱抗生素的广泛使用，颌面部感染病原菌耐药的情况越来越普遍。

### 一、口腔颌面部感染的途径及病原菌
#### （一）口腔颌面部感染的途径
`牙医腺血有损伤`
1. ==牙源性==：**最主要途径**，大多由==需氧和厌氧菌混合感染==造成。
2. ==腺源性==：面颈部淋巴结继发于口腔、上呼吸道感染；还可引起筋膜间隙的蜂窝织炎；最常见的腺源性感染：颌下间隙感染。
3. ==损伤性==
4. ==血源性==
5. ==医源性==

#### （二）病原菌
因病原菌不同，口腔颌面部感染可分为**非特异性感染**和**特异性感染**两大类。
1. 非特异性感染（化脓性感染）的病原体
	（1）[[金黄色葡萄球菌]]（**主**）（混合型感染）
	（2）溶血性链球菌
	（3）大肠杆菌等
2. 特异性感染的病原体 `姐妹放风`
	（1）结核
	（2）梅毒
	（3）放线菌
	（4）破伤风
- 有些颌面部感染并非细菌性，可由病毒、真菌引起，如疱疹性口炎、念珠菌性口炎及 HIV 感染引起的口腔表现。

### 二、口腔颌面部感染的临床表现

1. 急性炎症期：①局部症状；②全身症状
2. 慢性炎症期：①局部症状；②全身症状
- 两两组合

#### （一）局部症状
1. 化脓急性期：
	（1）局部表现为**红、肿、热、痛和功能障碍、引流区淋巴结肿痛**等典型症状
	（2）炎症累及咀嚼肌可致不同程度==开口受限==；
	（3）病变位于==口底、舌根、咽旁==可有==进食、吞咽、言语和呼吸困难==。
	（4）腐败坏死性蜂窝组织炎的局部皮肤呈==弥漫性水肿==，紫红色或灰白色，无弹性，有明显==凹陷性水肿==，由于组织间有气体产生可触及**捻发音**。
2. 感染的慢性期
	（1）局部形成较硬的==炎性浸润块==，并出现不同程度的==功能障碍==。
	（2）有的脓肿形成未及时治疗而==自行破溃==，则形成==长期排脓的窦（瘘）口==
	（3）机体抵抗力减弱或治疗不彻底时，慢性感染==可再度急性发作==。

#### （二）全身症状
全身症状因细菌毒力及机体抵抗力不同而有差异，其表现有轻重之分。
==局部症状决定全身症状==：局部反应轻微的炎症可无全身症状；反之，局部反应较重的，全身症状也较明显
1. 急性期：
	（1）全身症状包括==畏寒、发热、头痛、全身不适、乏力、食欲减退、尿量减少==等，实验室检查可见**白细胞总数增高，中性粒细胞比例上升，核左移**；
	（2）病情较重而时间长者，由于代谢紊乱，可导致==水、电解质失衡、酸中毒==
	（3）严重感染伴==败血症或脓毒血症==时，可发生==中毒性休克==
	（4）患者全身反应底下，多器官功能衰竭，如脉搏微弱、血压下降、体温和白细胞计数不升高或低于正常时，均提示病情重笃。
2. 慢性期：
	（1）局部炎症久治不愈，长期排脓或反复发作，可伴有==持续低热==。因长期处于慢性消耗状态，患者可表现为==全身衰弱和营养不良==。

#选择题

### 三、口腔颌面部感染的诊断
炎症 + 脓肿 + 中毒 + 骨髓炎
1. 炎症初期，感染区的**红、肿、热、痛、功能障碍**等是主要表现，也是诊断局部感染的基本依据。
2. 浅部局限脓肿，**波动感**是诊断浅部脓肿的重要特征。
3. 深部脓肿，尤其是位于筋膜下层的脓肿，一般==难有波动感==， 但==压痛点比较明显==，按压脓肿部的表面皮肤常出现**凹陷性水肿**。可用 [[第2单元 口腔颌面外科基本知识及基本操作#（二）穿刺检查|穿刺法]] 协助诊断或借助 B 超或 CT。
4. 进行脓液涂片、 细菌培养及药敏试验，以选择合适的抗菌药物。如怀疑菌血症时，行多次血培养。
5. 中毒感染或大量抗菌药物应用下，白细胞可无明显增加，但有核左移及中毒性颗粒出现。
6. 影像学检查对**颌骨骨髓炎**的诊断、病变范围、死骨形成等都能提供可靠依据。

### 四、口腔颌面部感染的治疗
治疗两方面
1. **局部**（主）：①适用轻度感染②是疗效好坏的**关键**
2. **全身**（辅）：改变病人一般状况，改善机体抵抗力

#### 间隙感染严重程度的分级
感染的严重程度主要取决于感染的部位、波及范围以及是否有造成呼吸道阻塞的可能等因素。

| 严重程度 | 解剖位置                                       | 解释                              |
| :--: | ------------------------------------------ | ------------------------------- |
|  轻度  | 前庭区、腭部、皮下、颊间隙。                             | 仅涉及口腔前庭部或皮下                     |
|  中度  | 下颌下间隙、颏下间隙、舌下间隙、翼下颌间隙、咬肌间隙、颞浅间隙、颞深间隙、眶下间隙。 | 可能会阻塞气道并造成器官插管困难等（`如舌体抬高、会厌水肿`） |
|  重度  | 咽旁间隙、咽后间隙、气管前间隙                            | 直接挤压呼吸道，并损伤重要的组成结构，`如侵入颅内等`。    |
|  极重  | 纵隔、颅内感染                                    |                                 |

#### （一）局部治疗
1. 保持局部清洁，减少局部活动度，避免不良刺激，特别对面部**疖、痈应严禁挤压**，以防感染扩散。
2. 急性期外敷中草药可起到散瘀、消肿、止痛或促进炎症局限的作用；已有局限倾向时，促使炎症消散或加速形成脓肿及排脓

#### （二）手术治疗
##### 1. 脓肿切开引流术 #简答
1. 切开引流的**目的**
	（1）==消炎解毒==，使脓液和腐败坏死物迅速排出体外
	（2）解除局部疼痛、肿胀及张力，==以防发生窒息==（如舌根部、口底间隙脓肿）
	（3）颌骨周围脓肿引流，以免并发==边缘性骨髓炎==
	（4）==预防感染向颅内和胸腔扩散或侵入血循环==，并发==严重并发症==。

2. 切开引流的**指征**
	（1）局部疼痛加重，并呈搏动性跳痛；炎性肿胀明显，皮肤表面紧张、发红、光亮；触诊时有明显压痛点、波动感，呈凹陷性水肿；或深部脓肿经**穿刺有脓液**抽出者。
	（2）口腔颌面部急性化脓性炎症，==经抗菌药物控制感染感染无效==，同时出现明显的全身中毒症状者。
	（3）==儿童==颌周蜂窝组织炎，如炎症已累及多间隙，出现呼吸困难或吞咽困难，可以==早期切开减压==，以迅速缓解呼吸困难及防止炎症继续扩散。
	（4）**结核性**淋巴结炎，经局部及全身抗结合治疗无效，皮肤发红已近自溃的脓肿，必要时可行==脓肿切开==。（一般不切，窦道不易愈合）

3. 切开引流的**要求**（开放式的） #简答
	（1）为达到体位自然引流的目的，切口位置应在脓腔==重力的低位==。
	（2）切口应尽量选择在愈合后==瘢痕隐蔽==的位置，保证==引流通畅==为原则；一般首选经==口内引流==。颜面部脓肿应沿皮纹方向切开，勿损伤重要解剖结构，如面神经、血管和唾液腺导管等。
	（3）一般切开至==黏膜下或皮下==即可，按脓肿位置用血管钳直入脓腔后再用钝分离扩大创口。避免在不同组织层次中形成多处脓腔或通道，减少感染扩散，保证引流通畅。（==避免二次分离==）
	（4）手术操作==应准确轻柔==；颜面危险三角区脓肿切开后，**严禁挤压**。

4. 建立引流
	（1）一般口内用==碘仿纱条或橡皮片==引流。
	（2）口外脓肿可用==盐水纱条或橡皮片、乳胶管==固定。
	（3）每天更换敷料 1~2 次；必要时需要 1%~3% 过氧化氢溶液、生理盐水或抗菌药物液冲洗。

##### 2. 清除病灶：去除病灶牙

#### （三）全身治疗（对因治疗和对症治疗）

1. 口腔颌面部感染并发全身中毒症状，都应在==局部处理的同时，给予全身支持治疗==，并及时有针对性地给予抗菌药物。（**局部为主，全身为辅**）
2. 避免长期、无针对性的大剂量==广谱抗菌药物的滥用==→耐药菌株的增加，二次感染。
3. 根据病原种类及==药物敏感试验==结果选用抗菌药物。
4. 遵循==口服—肌注—静脉==的顺序用药。
5. 严格联合用药指征，==能单一不联合==。

| 抗菌药物的基本原则（**尽量少用**）  |
| -------------------- |
| ==用药前应尽可能进行药敏试验==    |
| 能用==窄谱者不用广谱==        |
| 遵循==口服、肌注、静脉==的顺序    |
| 适当的用药指征和剂量           |
| 严格联合应用指征，==能单一就不联合== |
| 恰当地预防性用药             |
| 注意：感染治疗以局部为主，全身治疗为辅。 |

## 第二节 智齿冠周炎（`pericoronitis`）
**智齿冠周炎**（`pericoronitis`）**定义**：是指智齿（第三磨牙）萌出不全或阻生时，牙冠周围软组织发生的炎症，临床上以下颌智齿冠周炎多见。 #名解

![[图5-2 下颌智齿冠周炎扩散途径.png]]

### 一、病因
1. 咀嚼器官退化，造成颌骨长度与牙列所需长度的不协调，牙量＞骨量。
2. 下颌第三磨牙最后萌出，因萌出位置不足，可导致程度不同的阻生。
3. 智齿在萌出过程中，牙冠可部分或全部为龈瓣覆盖，龈瓣与牙冠之间 形成较深的盲袋，食物及细菌极易嵌塞于盲袋内。
4. 冠部牙龈常因咀嚼食物而损伤，形成溃疡。
5. 当全身抵抗力下降、局部细菌毒力增强时，可引起冠周炎急性发作。

### 二、临床表现
1. 局部急性炎症
	(1) 患者自觉患侧磨牙后区胀痛不适，进食咀嚼、吞咽及开口运动时疼痛加重。病情发展，局部可呈自发性跳痛或放射性痛。
	(2) 若炎症侵及咀嚼肌，可引起肌的反射性痉挛而出现不同程度开口受限，甚至出现“牙关紧闭”。
	(3) 口腔不洁，出现口臭、舌苔变厚、患牙龈袋处有咸味分泌物溢出。
2. 全身症状
	(1) 畏寒、发热、头痛、食欲减退、大便秘结等。
	(2) 可见白细胞总数增高，中性粒细胞比例上升。
3. 慢性冠周炎
	(1) 多无明显症状，仅出现局部轻度压痛、不适。
4. 口腔局部检查
	(1) 探针探查—智齿周围软组织及牙龈红肿，龈瓣边缘糜烂，有明显触痛，龈袋内可挤压出脓液。
	(2) 病情严重者，炎性肿胀可波及腭舌弓和咽侧壁，并伴开口受限。
	(3) 化脓性炎症局限后，可形成冠周脓肿，有时可自行溃破。
	(4) 患侧下颌淋巴结肿胀、压痛。
5. 冠周炎蔓延**扩散途径** #简答
	(1) 向磨牙后区扩散——骨膜下脓肿——==咬肌前缘==与==颊肌后缘==间的薄弱处发生皮下脓肿——**面颊瘘**。
	(2) 沿下颌骨外斜线向前，可在相当于**下颌第一磨牙颊侧黏膜转折处**的骨膜下形成脓肿或破溃成瘘。
	(3) 沿下颌支==外==侧——**咬肌间隙感染**。
	(4) 沿下颌支==内==侧——**翼下颌间隙感染**。
![[图5-4 智齿冠周炎的感染扩散途径.png|661]]

### 三、诊断
1. 病史 + 临床症状 + 检查
2. X 线检查
3. 合并==第一磨牙颊侧瘘==时，要注意鉴别（68 之间插入诊断丝）
4. 疼痛应与==下 7==远中深龋引起的牙髓炎及==下 8==区牙龈恶性肿瘤相鉴别。

### 四、治疗

早期诊断及及时治疗非常重要
1. 治疗原则是急性期消炎、镇痛、切开引流、增强全身抵抗力为主。
2. 当炎症转入慢性期后，若为不可能萌出的阻生牙应尽早拔除。

#### （一）急性期

1. 局部冲洗：生理盐水、1%~3% 过氧化氢溶液,0.1% 氯己定溶液等。
2. 选择抗菌药物及全身支持治疗
3. ==切开引流==：如龈瓣附近形成脓肿，应及时切开并放置引流条。

#### （二）慢性期

1. ==冠周龈瓣切除术==
	（1）急性炎症消退后，对有足够萌出位置且牙位正常的智齿，可在局麻下行冠周龈瓣切除术，以消除盲袋。
2. ==下颌智齿拔除术==
	（1）牙位不正；无足够萌出位置
	（2）相对的第三磨牙位置不正或已拔除者，以及为避免冠周炎的复发，均应及早拔除
	（3）伴有面瘘者，在拔牙的同时应切除瘘管，刮净肉芽，缝合瘘道

## 第三节 口腔颌面部间隙感染

1. 感染部位：==阻力薄弱的疏松结缔组织，是感染发生和扩散的潜在间隙==。
2. 感染来源：全部为**继发性**——常见为==牙源性==或==腺源性==感染扩散所致，损伤性、医源性、血源性较少见。
3. 感染细菌：需氧菌和厌氧菌引起的混合感染（主要）。葡萄球菌和链球菌引起的**化脓性感染**或厌氧菌引起的**腐败菌感染**。
4. 感染进程：蜂窝织炎——脂肪结缔组织变性坏死——脓肿。
5. 感染特点：化脓性炎症可局限于一个间隙内，也可波及相邻几个间隙，甚至可沿神经、血管扩散，引起海绵窦血栓性静脉炎、脑脓肿、败血症、纵隔炎等严重并发症。

### （一）眶下间隙感染（`infraobital space`）

1. 感染位置：**眶下方**、上颌骨前壁与面部表情肌之间。
	- 内容：间隙中有从眶下孔，穿出之眶下神经、血管以及眶下淋巴结。内眦动脉、面静脉及其与眼静脉、眶下静脉、面深静脉的交通支。
2. 感染来源
   （1）==上颌尖牙及第一前磨牙或上颌切牙 1～4== 的根尖化脓性炎症或牙槽脓肿。
   （2）==上颌骨骨髓炎==。
   （3）==上唇底部与鼻侧的==化脓性炎症。
3. 临床特点：
	（1）**肿胀**：眶下区肿胀范围波及内眦、眼睑、额部皮肤。肿胀区皮肤发红、张力增大，眼睑水肿、睑裂变窄、鼻唇沟消失。
	（2）**波动感**：眶下区可触及==波动感==。口腔前庭龈颊沟处常有明显肿胀、压痛波动感。
	（3）**疼痛、无受限**：肿胀及炎症激惹==眶下神经==，可引起不同程度的==疼痛==。（==无张口受限==)
	（4）**扩散**：感染向框内直接扩散，形成==眶内蜂窝织炎==。沿着面静脉、内眦静脉、眼静脉向颅内扩散，可并发==海绵窦血栓性静脉炎==。
4. 治疗：
	（1）**抗菌**：眶下间隙蜂窝织炎阶段可全身应用==抗菌药物==；
	（2）**切开引流**：一旦脓肿形成，应及时作==切开引流==。
	（3）**低位引流**：按==低位引流==原则，常在口内上颌尖牙及前磨牙唇侧口腔前庭黏膜转折处作切口
	（4）==横行切开==黏骨膜达骨面，用血管钳向尖牙窝方向==分离脓肿==，使脓液充分引流。

### （二）颊间隙感染（`temporal space`）

1. 感染位置
   （1）**广义**：指位于==颊部皮肤==和==颊黏膜==之间==颊肌周围的间隙==。
   （2）**狭义**：指位于==咬肌和颊肌==之间存在的一个狭小筋膜间隙。
2. 感染来源
   （1）**根尖周脓肿或牙槽脓肿**：上、下颌磨牙的根尖周脓肿或牙槽脓肿穿破骨膜，侵入颊间隙。
   （2）**损伤**：因颊部==皮肤损伤==、==颊黏膜溃疡==继发感染。
   （3）**淋巴结炎症**：==颊、颌上淋巴结==的炎症扩散。
3. 临床特点：
	- 脓肿范围与是否波及**颊脂垫**有关
	（1）**未波及颊脂垫**：在颊部==皮下或黏膜下脓肿==，病程进展==缓慢==，范围较为==局限==。
	（2）**波及颊脂垫**：感染波及**颊脂垫**时，则炎症发展==迅速==，肿胀范围波及==整个颊部==，并可向相邻间隙==扩散==，形成==多间隙感染==。
4. 治疗
   （1）==口内切口==：应在脓肿低位，即口腔前庭、下颌龈颊沟之上切开。
	（2）==口外切口==：①颊部皮下脓肿可在脓肿浅表皮肤沿皮肤皱折线切开。②广泛颊间隙感染则应从**下颌骨下缘以下 1~2 cm 处**（**唯一 1~2 cm**）作平行于下颌骨下缘的切口，从切开的皮下向上潜行钝分离，进入颊部脓腔。注意避免损伤==面神经下颌缘支及面动脉、面静脉==等。

### （三）颞间隙感染（temporal space）

1. 感染位置：位于颧弓上方的颞区，借==颞肌==分为**颞浅**与**颞深**两个间隙；借脂肪结缔组织与 [[颞下间隙]]、[[咬肌间隙]]、[[翼下颌间隙]]、[[颊间隙]] 相通。
2. 感染来源
   （1）**扩散**：常由咬肌间隙、翼下颌间隙、颞下间隙、颊间隙感染扩散引起。
   （2）**耳源性感染**（化脓性中耳炎、颞乳突炎）
   （3）**疖痈、损伤**：颞部疖痈、颞部损伤继发感染也可波及颞间隙
3. 临床特点：
	（1）**范围**：==单纯颞间隙感染==，肿胀范围可仅局限于颞部。==伴有相邻多间隙感染==，同时有腮腺咬肌区、颊部、眶部、颞部等区广泛肿胀。
	（2）**水肿、疼痛、受限**：病变区有==凹陷性水肿，压痛、咀嚼痛==和不同程度的==开口受限==。
	（3）**波动感**：脓肿形成后,==颞浅间隙脓肿可触及波动感==，==颞深间隙脓肿==则需借助影像学检查或穿刺抽出脓液才能明确诊断。
	（4）**颞骨骨髓炎**：颞肌坚厚，颞筋膜致密，深部脓肿难以自行穿破，脓液长期积存于颞骨骨表面，可引起==颞骨骨髓炎==。
	（5）**并发症**：颞骨鳞部骨壁薄，内、外骨板间板障少，感染可直接从骨缝或通过进入脑膜的血管蔓延，导致脑膜炎、脑脓肿等并发症。
4. 治疗
	（1）**浅部脓肿**可在颞部发际内作==单个皮肤切口==;
	（2）**深部脓肿**可作==两个以上与颞肌纤维方向一致的直切口==;
	（3）当疑有颞骨骨髓炎时，可沿颞肌附着作弧形皮肤切口，切开颞肌附着，由骨面翻起颞肌，使颞鳞部完全敞开引流。注意行弧形切口时，**切忌**与肌纤维相交的横行切口，因为切断颞肌可损伤颞肌的神经、血管，破坏颞肌的功能。
	（4）多间隙感染，还应在下颌下区另作切口，行==上下贯通式引流==。

### （五）咬肌间隙感染 (`masseteric space`)

1. 感染位置：位于咬肌与下颌支外侧骨壁之间
	- 咬肌在下颌支及其角部附着宽广紧密，潜在性咬肌间隙存在于==下颌支上段的外侧部位==。是==最常见==的颌面部感染之一。
2. 感染来源
   （1）主要来自下颌智齿冠周炎，下颌磨牙的根尖周炎、牙槽脓肿。
   （2）因相邻间隙如颞下间隙感染的扩散
   （3）偶有因化脓性腮腺炎波及者。
3. 临床特点
   （1）**肿胀、受限**：典型症状是以下颌支及下颌角为中心的咬肌区==肿胀==、变硬、压痛伴明显==开口受限==。
   （2）**无波动感**：由于咬肌肥厚坚实，脓肿难以自行溃破，也==不易触到波动感==。
   （3）**凹陷性水肿**：若炎症持续 1 周以上，==压痛点局限或有凹陷性水肿==，穿刺有脓液时，应行切开引流; 否则，由于长期脓液蓄积，易形成下颌支的==边缘性骨髓炎==。
4. 治疗：
	（1）全身应用抗菌药物；局部物理疗法
	（2）一旦脓肿形成，及时引流，临床常用**口外**途径切开引流：口外切口从下颌支后缘绕过下颌角，距下颌下缘 **2 cm**（或 1.5~2 cm） 处切开，切口长约 3~5 cm。逐层切开皮下组织、颈阔肌以及咬肌在下颌角区的部分附着，用骨膜剥离器由骨面推起咬肌进入脓腔，引出脓液冲洗脓腔后，建立充分引流 `（因为口内切口在翼下颌皱襞外侧。然需要进入下颌升支外侧，且患者张口受限）`
	（3）如有边缘性骨髓炎，在脓液减少后应早期==行病灶刮除术==。
	（4）咬肌间隙感染缓解或被控制后，应==及早==对引起感染的病灶牙进行治疗或拔除。

### （三）翼下颌间隙感染

1. 感染位置：下颌支与翼内肌之间。
2. 感染来源
   （1）下牙槽神经阻滞麻醉时消毒不严。
   （2）下颌智齿冠周炎及下颌磨牙根尖周炎。
   （3）相邻间隙。
3. 临床特点
   （1）肿胀部位：下颌升支后缘稍内侧、翼下颌皱襞处黏膜水肿。
   （2）张口受限：有。
   （3）波动感：无（穿刺诊断）。
4. 治疗：切口：翼颌皱襞稍外侧纵行切开 2～3 cm。口外：同咬肌间隙感染。

### （四）颞下颌间隙感染

1. 感染位置：颅中窝底。
2. 感染来源
   （1）相邻间隙。
   （2）上颌结节阻滞麻醉时带入。
   （3）上颌磨牙的根尖周感染或拔牙后感染引起。
3. 临床特点
   （1）肿胀部位：颞弓上、下及下颌支后方微肿。
   （2）张口受限：有。
   （3）波动感：无（穿刺诊断）。
4. 治疗：口外切口同咬肌间隙。

### （五）下颌下间隙感染

1. 感染位置：下颌下三角内。
2. 感染来源
   （1）下颌智齿冠周炎、下颌后牙根尖周炎、牙槽脓肿等牙源性（多见）。
   （2）下颌下淋巴结炎。
   （3）化脓性下颌下腺炎。
3. 临床特点
   （1）肿胀部位：下颌下三角区肿胀，下颌骨下缘轮廓消失。
   （2）波动感：有。
4. 治疗：下颌骨体部下缘以下 2 cm 作与下颌下缘平行之切口。

### （八）咽旁间隙感染

1. 感染位置：咽腔侧方的咽上缩肌与翼内肌和腮腺深叶之间。
2. 感染来源
   （1）下颌智齿冠周炎。
   （2）腭扁桃体。
   （3）相邻间隙。
   （4）腮腺炎、耳源性炎症和颈深上淋巴结炎。
3. 临床特点
   （1）肿胀部位：咽侧壁红肿、悬雍垂推向健侧。
   （2）张口受限：有。
   （3）波动感：无（需穿刺）。
4. 治疗
   （1）切口：口内—翼下颌皱襞稍内侧纵行切开黏膜层（首选）。
   （2）口外—同咬肌间隙。

### （九）口底多间隙感染

腐败坏死性口底多间隙感染称为路德维希咽峡炎。

1. 感染位置：双侧下颌下、舌下以及颊下间隙感染来源。
2. 感染来源
   （1）下颌牙各种炎症。
   （2）下颌下腺炎、淋巴结炎。
   （3）急性扁桃体炎、口底软组织和颌骨的损伤。
3. 临床特点
   （1）化脓性：一侧下颌下间隙或舌下间隙肿胀，之后弥漫性肿胀、凹陷性水肿。
   （2）腐败坏死性：肿胀皮肤呈紫红色，压痛，凹陷性水肿明显，扪及捻发音。
   （3）切开后有大量咖啡色、稀薄、恶臭、混有气泡的液体。
4. 治疗：广泛切开——“衣领”型或倒“T”型切口。

## 第四节 颌骨骨髓炎

### 一、化脓性颌骨骨髓炎

好发年龄：青壮年（16～30 岁最多）。性别：男多，约 2:1。化脓性颌骨骨髓炎约占各类型颌骨骨髓炎的 90% 以上。好发部位：下颌骨。

#### （一）病因与分类

1. 病原菌：金黄色葡萄球菌，临床多是混合性细菌感染。
2. 感染途径
   （1）牙源性感染（最多见，约占 90%）。
   （2）损伤性感染。
   （3）血行性感染（多见于儿童）。
3. 分类：中央性和边缘性。

#### （二）临床表现

1. 颌骨骨髓炎的临床发展过程：可分为急性期和慢性期两个阶段。
   （1）急性期
       全身：发热、寒战、疲倦无力、食欲缺乏，白细胞↑，中性粒细胞↑。
       局部：肿胀、充血，病源牙可有明显叩痛及伸长感。
   （2）慢性期
       全身：症状轻。体温正常或仅有低热；消瘦、贫血，机体慢性中毒消耗症状。
       局部：肿胀，微红，出现多个瘘孔、牙松动。
2. 中央性颌骨骨髓炎：多由根尖周脓肿发展而来。
   （1）急性期
       ① 全身症状
          初期：全身寒战、发热，体温可达 40℃；白细胞可达 20×10^9/L 以上；食欲减退，嗜睡。
          化脓期：中毒症状及局部症状加重，可引起败血症。
       ② 局部症状（肿痛，淋巴结肿大）
          变区牙剧痛、松动，向半侧颌骨或三叉神经分支区放射。
          龈袋溢脓，龈充血水肿。
          下唇麻木。
          张口受限（波及下颌支、髁突及喙突，激惹翼内肌、咬肌所致）。
   （2）慢性期：发病两周可转为慢性期。
       局部：瘘孔、长期排脓，可有大块死骨——形成病理性骨折——咬合错乱、面部畸形。
3. 边缘性颌骨骨髓炎
   好发部位：下颌骨（下颌升支及下颌角）。
   增生型：青年人，抵抗力强者，骨密质增生，骨质呈致密影像。
   溶解破坏型：抵抗力较弱者，有瘘孔，骨质稀疏脱钙，骨粗糙面，少有大块死骨。

#### （三）诊断

1. 诊断依据
   （1）急性中央性：病原牙松动，患侧下唇麻木（下颌骨）。上颌骨骨髓炎波及上颌窦时，患侧的鼻腔溢脓。
   （2）慢性中央性：瘘管形成和溢脓。
   （3）边缘性颌骨骨髓炎：类似间隙感染。
   X 线：溶解破坏型—骨小梁排列紊乱与死骨形成；增生型—骨膜反应性增生。
2. 中央性颌骨骨髓炎与边缘性颌骨骨髓炎的鉴别

| 项目 | 中央性颌骨骨髓炎 | 边缘性颌骨骨髓炎 |
|:---|:---|:---|
| 感染来源 | 牙周膜炎、根尖周炎 | 下颌智齿冠周炎 |
| 感染途径 | 先松质骨，后密质骨 | 骨膜，密质骨 |
| 临床表现 | 弥漫型较多 | 局限型较多 |
| 累及牙是否松动 | 是 | 否 |
| 病变部位 | 多在颌骨体 | 多在下颌角及升支 |
| X 线 | 大块死骨形成，界清 | 增生型：骨密质增生（骨膜反应）<br>溶解破坏型：形成不均匀小块的骨粗糙面 |
| 急性转慢性时间 | 2 周 | — |
| 骨质破坏时间 | — | 2～4 周，儿童 7～10 天 |
| 手术时间 | 病变局限 3～4 周<br>病变弥散 5～6 周 | 慢性期 2～4 周 |

#### （四）治疗原则

1. 急性期
   （1）治疗原则：同一般急性炎症，全身治疗 + 外科手术治疗。
   （2）治疗方法：
       ① 引流排脓及除去病灶。
       ② 中央性颌骨骨髓炎中，及早拔除病灶牙及相邻的松动牙（甚至骨板）。
2. 慢性期
   （1）手术去除死骨和病灶。
       ① 中央性：摘除死骨为主。
       ② 边缘性：有死骨刮死骨，没死骨刮肉芽。
   （2）死骨摘除手术指征：经久不愈的瘘管、有死骨、患者可耐受手术者。
   （3）死骨摘除手术时间：
       ① 中央性：病变局限者 3～4 周；病变弥漫者 5～6 周。
       ② 边缘性：2～4 周。

### 二、新生儿颌骨骨髓炎

#### （一）概念

1. 年龄：出生后 3 个月以内。
2. 类型：化脓性中央性颌骨骨髓炎。
3. 部位：上颌骨。

#### （二）病因

1. 感染途径：血源性。
2. 病因：牙龈损伤；母亲化脓性乳腺炎；患泪囊炎或鼻泪管炎。
3. 致病菌：金黄色葡萄球菌。

#### （三）临床表现与诊断要点

1. 全身症状：同一般骨髓炎。
2. 局部症状：眶下及内眦部皮肤红肿——眶周蜂窝组织炎。颗粒状死骨、大块死骨少见（因上颌骨骨质松软、血供丰富）。

#### （四）治疗原则（保守）

1. 大量有效抗生素治疗。
2. 若形成脓肿，要及早切开引流。
3. 换药时，最好用青霉素等抗生素溶液冲洗，效果较好。
4. 口内有瘘孔者应注意防止脓液误吸引起肺部并发症。
5. 如病情转入慢性期
   （1）较小死骨，自排。
   （2）未感染的牙胚要尽量保留。
   （3）较大死骨不能排出者亦尽量保守——避免颌面畸形或咬合紊乱。

### 三、放射性颌骨坏死

口腔软组织射线耐受量 6～8 周，60～80 Gy。

#### （一）临床表现与诊断要点

1. 发病时间：放射治疗后，数月乃至十余年。
2. 局部症状：持续性针刺样剧痛。
3. 主要特征：死骨与正常骨界限不清。
4. 全身症状：慢性消耗性衰竭，消瘦及贫血。

#### （二）治疗原则

1. 全身治疗：抗生素、镇痛剂、输血、高压氧促进死骨分离。
2. 局部治疗：死骨未分离前，低浓度 H₂O₂ 或抗生素液进行冲洗。
3. 死骨形成：骨钳分次逐步咬除（健康骨质范围内施行）。

#### （三）预防

1. 放疗前（7～10 天）：洁治、该治的治、该拔的拔、该拆的拆（拆除金属修复体）。
2. 放疗中
   （1）溃疡：涂抗生素软膏。
   （2）氟化物防龋。
   （3）对非照射区隔离保护。
3. 放疗后（3～5 年）
   （1）禁忌拔牙，必须拔牙时，术前、术后均应使用有效的抗生素，避免感染。
   （2）“精确”放疗。

## 第五节 面部疖痈

### 一、概念

1. 疖（单一毛囊）、痈（相邻多数毛囊）及其附件的急性化脓性炎症。
2. 致病菌：金黄色葡萄球菌。

### 二、临床表现

1. 疖：红、肿、痛的小硬结，一个脓头，症状较轻。
2. 痈：上唇多见，男性多见。
3. 黏膜上出现多数的黄白色脓头，局部淋巴结肿大、压痛，全身中毒症状明显。

### 三、并发症

疖痈最易发生全身并发症。原因：
① 病毒菌毒力强。
② “危险三角区”内静脉常无瓣膜。
③ 颜面表情肌肉活动。

### 四、治疗原则（常考）

1. 禁忌：挤压、挑刺、热敷或用苯酚、硝酸银烧灼（防感染扩散）。
2. 治疗方法
   （1）疖：2% 碘酊涂擦。
   （2）痈：
       ① 高渗盐水或含抗生素的盐水纱布持续湿敷。
       ② 皮下脓肿久不破溃：保守性切开，切忌分离脓腔。
       ③ 破溃或切开后：高渗盐水湿敷。
       ④ 脓栓不排出：消毒镊取出。

## 第六节 面颈部淋巴结炎

### 一、病因

1. 牙源性及口腔感染（最多见）。
2. 皮肤的损伤、疖痈。
3. 小儿患者：上呼吸道感染及扁桃体炎引起。
4. 化脓性淋巴结炎：葡萄球菌及链球菌。
5. 结核性淋巴结炎：结核杆菌。

### 二、临床表现

1. 化脓性淋巴结炎：可分为急性和慢性两类。
   （1）急性
       ① 浆液期：局部症状—淋巴结肿大、疼痛、可移动，界清，无粘连；全身反应—甚微或有低热。
       ② 化脓期：局部症状—疼痛加重，局部组织充血、肿、硬，淋巴结粘连不动；全身症状—重，高热、寒战、头痛、全身无力、食欲减退，小儿可烦躁不安。
   （2）慢性
       ① 局部：淋巴结内结缔组织增生形成微痛的硬结。
       ② 全身：症状不明显，若机体抵抗力下降，炎症可急性发作。
2. 结核性淋巴结炎
   （1）发病年龄：儿童及青年。
   （2）临床表现：淋巴结中心因有干酪样坏死似米汤、冷脓肿。

### 三、诊断

根据脓液进行鉴别诊断。结核的脓液：稀薄污浊，暗灰色似米汤，干酪样坏死物。

### 四、治疗

常用抗结核药物：异烟肼、利福平、链霉素、乙胺丁醇、吡嗪酰胺。

## 第七节 颌面部特异性感染

### 一、颌面骨结核

1. 感染来源：血行播散、牙龈创口感染、口腔黏膜及牙龈结核直接累及。
2. 发病年龄：儿童、青少年。
3. 好发部位：上颌骨颧骨结合部和下颌支。
4. 临床特征：干酪样坏死物，冷脓肿。
5. 诊断：涂片可见抗酸杆菌。
6. 治疗：全身支持、营养疗法和抗结核治疗。

### 二、颌面部放线菌病

1. 致病菌：Wolff-Israel 型放线菌。
2. 好发年龄：20～45 岁，男性多见。
3. 临床特点
   （1）腮腺咬肌区。
   （2）表面皮肤呈棕红色。
   （3）黄色黏稠脓液，硫磺样颗粒。
   （4）涂片：革兰阳性、呈放射状的菌丝。
4. 药物治疗
   （1）青霉素首选。
   （2）碘制剂。
   （3）免疫疗法。
5. 手术疗法
   （1）切开引流。
   （2）死骨刮除术。
   （3）病灶切除术。

### 三、颌面部先天梅毒

哈钦森三征：梅毒性间质性角膜炎致角膜混浊、第Ⅷ对脑神经损害所致神经聋以及哈钦森牙，被称为先天性梅毒的三征。
