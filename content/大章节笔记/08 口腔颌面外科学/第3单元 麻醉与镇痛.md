# 第三单元 麻醉与镇痛

**麻醉**（anesthesia)：是指用药物或其他方法使患者整体或局部暂时失去感觉以减少或消除患者在口腔治疗或手术中的疼痛。
**麻醉学**（anesthesiology)：研究消除患者手术疼痛，保证患者安全，为手术创造良好条件的一门学科。
#名解

## 第一节 局部麻醉

局部麻醉 (local anesthesia)：是指用局部麻醉药暂时性阻断机体一定区域内神经末梢和纤维的感觉传导，从而使该区疼痛消失的方法。 #名解
1. 优点：①痛觉消失，温度觉、触压觉存在；②术者可独立操作不需要麻醉医师；③患者意识清醒；④可以与血管收缩剂合用，减少出血；
2. 不足：不适用于==不合作患儿==和局部有==炎症==者

### 一、局部麻醉药物

#### （一）分类

| 分类          | 代表药                            |
| :---------- | :----------------------------- |
| 酯类（**易过敏**） | 普鲁卡因、地卡因（丁卡因）                  |
| 酰胺类（常用）     | 利多卡因、布比卡因（丁哌卡因）、阿替卡因、甲哌卡因、罗哌卡因 |

#### （二）各药物特点

1. 普鲁卡因（奴佛卡因、novocaine）
   （1）优点：安全、毒性和副作用**小**；
   （2）缺点：偶能产生==过敏反应==，不适用于==表面麻醉==，不和==磺胺类抗生素==合用，易==出血==（不表、不磺）
   （3）一次最大剂量 1000 mg（6.0 mg/kg）；2% 浓度最大用量 50 mL。（浓度与用量负相关）
   （4）维持时间 45～60 min。
   - 效能、毒性都看作 1，其它药物与其对比。

2. 利多卡因 (lidocaine)（赛洛卡因，xylocaine）
   （1）优点：①局麻作用较普鲁卡因**强**；②具有安全而迅速的**抗室性心律失常作用**（心律失常患者常作为首选）。
   （2）缺点：毒性较普鲁卡因**大**。
   （3）一次最大剂量 300～400 mg（4.4 mg/kg），2% 浓度最大用量 20 mL。
   （4）维持时间 90～120 min。

3. 布比卡因（麻卡因、丁哌卡因、marcaine）
   （1）优点：持续时间为利多 ==2 倍==，强度 ==3-4 倍==。持续时间 **6 h** 以上，适合费时较长的手术和术后镇痛。
   （2）缺点：毒性**大**，无表麻作用。

4. 丁卡因（地卡因、pantocaine)
   （1）优点：穿透力**强**
   （2）缺点：毒性**大**、主要用作表面麻醉。
   （2）一次最大剂量 40～60 mg。

5. 阿替卡因（碧蓝麻）：
	（1）特点：穿透性扩散性强，用于成人和 **4 岁**以上儿童。
	（2）一次最大剂量 成人 7.0 mg/kg，儿童 5.0 mg/kg

6. 甲哌卡因：
	（1）特点：**3 岁**以下禁用，用于时间较短的治疗和==高血压患者==。（相对安全的药物，三甲）

| 药物         | 剂量                                      | 特点                            |
| ---------- | --------------------------------------- | ----------------------------- |
| 利多卡因       | 300～400 mg（4.4 mg/kg），2% 溶液 20 ml       | 迅速而安全的抗室性心律失常作用               |
| 布比卡因（丁哌卡因） | —                                       | 维持时间长，约 **6 h**               |
| 丁卡因        | 40～60 mg                                | 只用于表面麻醉                       |
| 甲哌卡因       | 150 mg                                  | **3 岁**以下儿童禁用；用于时间较短的治疗和高血压患者 |
| 阿替卡因       | 成人 7.0 mg/kg 儿童 5 mg/kg<br>（最大 200 mg） | **4 岁**以下儿童禁用                 |

#### （三）局麻药物过敏试验

1. 易发生过敏的药物：酯类：==普鲁卡因==。
2. 酯类易过敏的原因：①血浆内药物被==假胆碱酯酶==分解成==对氨基苯甲酸 (PABA)==和==乙二胺乙醇==。②药物中防腐剂：==对羟基苯甲酸甲酯==。
3. 普鲁卡因、利多卡因皮内试验：将 ==0.1 ml== 局麻药（1% 普鲁卡因或 2% 利多卡因）稀释至 ==1 ml==，皮内注射 0.1 ml，20 分钟后观察反应。局部红肿，**红晕直径超过 1 cm 者为阳性**（+）。

#### （四）血管收缩剂应用

1. 常用血管收缩剂：**肾上腺素**、盐酸苯肾上腺素（新福林）、渥克他加压素
2. 血管收缩剂作用 #简答
	① ==加强镇痛==效果。
	②==延缓==局麻药物==吸收==
	③ ==延长==局麻==时间==。
	④ ==降低毒性==反应。
	⑤ ==减少==术区==出血==，使术野清晰。
3. 加入浓度一般量： ==1:50000～1:200000==。
	以利多卡因为例：健康人一般注射 1：10000 肾上腺素，一次最大量 20 ml（0.2 mg 肾上腺素）。心血管疾病患者 4 ml（0.4 mg 肾上腺素）
4. 头痛 → 肾上腺素反应。

### 二、常用局部麻醉的方法

表麻麻醉、浸润麻醉、阻滞麻醉、冷冻麻醉

#### （一）表面麻醉 (superficial or topical anesthesia)

**定义**：是将麻醉剂涂布或喷射于手术区表面，麻醉药物被吸收而使末梢神经麻痹，以达到**痛觉消失**的效果。 #名解
1. 适应证
   （1）表浅的**黏膜下**脓肿切开引流。
   （2）拔除松动的乳恒牙。
   （3）行气管内插管前的黏膜表面麻醉。

2. 常用药物浓度
   （1）0.25%～0.5% 丁卡因。（毒性大，一般只用于表麻）
   （2）2%～5% 利多卡因。
    - 普鲁卡因、布比卡因无表麻作用

#### （二）冷冻麻醉 (frozen anesthesia)
**定义**：应用药物使局部组织迅速散热、温度骤然降低、以致局部感觉、**首先是痛觉消失**而达到暂时性麻醉的效果 #名解
1. 常用药物：**氯乙烷**。
2. 持续时间：3～5 min。
3. 适应证：①**黏膜下和皮下**浅表脓肿的切开引流，②拔除松动的乳恒牙。
4. 注意事项：麻醉区周围的皮肤、黏膜应涂布凡士林加以保护。

#### （三）浸润麻醉 (infiltration anesthesia）

**定义**：将局麻药注入组织内，以作用于==神经末梢==，使之失去传导痛觉的能力而产生麻醉效果。
   1. ==骨膜上浸润法==(supraperiosteal Infiltration）：
       （1）适应证：上颌或下颌前牙区的牙及牙槽突手术。（骨质较为疏松区域）
       （2）方法：
	       ①进针点—唇颊侧前庭沟；
	       ②进针角度—与黏膜成 45°；
	       ③进针深度—至触抵根尖平面的骨膜；
	       ④回抽无血，麻药剂量—0.5～1 mL。
   2. ==牙周膜注射法==(periodontal membrane Injection）：
      （1）**适应证**：①血友病等出血倾向疾病的患者，黏膜和阻滞麻醉效果不好时（追加麻醉）。
      （2）方法：
			①进针点—牙的近中或远中
			②进针深度：0.5 cm
			③麻药剂量：0.2 ml
      （2）优点：==损伤小==。
      （3）缺点：==疼痛==。
      - **计算机控制局部麻醉**(computer controlled local anesthesia）通过一种电动的且带有预设程序的局麻输注设备完成的麻醉。
			优点：保证一个特定的注药速率和可控制的压力，减少患者疼痛感和组织反应。

#### （四）阻滞麻醉 (block anesthesia)

**定义**：将局麻药液注射到**神经干**或其主要分支附近，以阻断神经末梢传入的刺激，使被阻断的该神经分布区域产生麻醉效果。 #名解
1. 优点：①麻醉效果好；②可减少麻药的==用量==和==注射次数==；③可减少疼痛和避免感染扩散；
2. 注意事项：①严格遵守无菌原则，以==防感染==；②注射麻药前，必须**回抽无血**，无回血注射麻药；③若见回血，退针少许；④改变方向后再行刺入，直到回抽无血时，方可注射麻醉药。

| 上颌神经（圆孔出颅，困难基本不用）<br>上牙槽后神经<br>腭前神经<br>鼻腭神经<br>眶下神经 | 下颌神经（卵圆孔出颅，困难基本不用）<br>下牙槽神经<br>舌神经<br>颊神经 |

| 注意事项 | 注意针尖刺入不宜**过深**，以免刺破上颌结节后方的==翼静脉丛==，引起**血肿**                 |

2. 眶下神经阻滞麻醉（眶下孔或眶下管注射法）(block anesthesia of infraorbital nerve)
	适应证：①同侧上颌切牙至前磨牙的拔除；②牙槽突修整及上颌囊肿剜除术；③唇裂整复术。
	方法：==口外法==、口内法
	（1）口外法：

| 口外法  | 眶下神经阻滞麻醉                                        |
| ---- | ----------------------------------------------- |
| 进针点  | 同侧鼻翼旁 1 cm                                      |
| 进针方向 | 与皮肤成 ==45°== 角，斜向==上、后、外==                      |
| 进针深度 | ==1.5 cm==                                      |
| 麻药剂量 | ==1 ml==                                        |
| 麻醉区域 | 同侧下睑、鼻、眶下区、上唇、上颌前牙、前磨牙，以及这些牙的唇侧或颊侧的牙槽骨、骨膜、牙龈和黏膜 |
| 注意事项 | 进入眶下管不可过深＜1 cm，以防损伤眼球                           |
|      | ![[图2-6 眶下神经阻滞麻醉口外注射法.png\|306]]                |

   （2）口内法：
       ① 进针点：==上颌侧切牙相应前庭沟==。
       ② 进针方向、角度：与上颌中线成 45° 角上、后、外。
       ③注意事项：不易进入眶下管，不常用

3. 腭前神经阻滞麻醉（腭大孔注射法）(block anesthesia of anterior palatine nerve)

| 体位   | 大张口，𬌗平面与地面成 ==60== 度角（大部分 45°）                    |
| 进针点  | 上 8 腭侧龈缘至腭中线弓形==凹面==连线的==中点==或者水平连线的中外 1/3（腭大孔标志处） |
| 进针方向 | 对侧口角，向==后上外==进针                                    |
| 进针深度 | ==0.3～0.5 cm==                                     |
| 剂量   | ==0.3～0.5 ml==                                     |
| 麻醉区域 | 同侧磨牙、双尖牙腭侧的黏骨膜，牙龈和牙槽骨**（3~8）**                     |
| 注意事项 | 注射点不可偏后，以免麻醉腭中、腭后神经，引起软腭、悬雍垂麻痹而致注射麻药不可过多，==恶心或呕吐== |

4. 鼻腭神经阻滞麻醉（腭前孔注射法）(block anesthesia of nasopalatine nerve)

6. 舌神经阻滞麻醉 (block anesthesia of lingual nerve)：
	（1）下牙槽神经阻滞麻醉触及骨壁后退回 **1 cm**，回抽无血注射 ==0.5～1 mL==。
	（2）麻醉区域：同侧下颌舌侧牙龈、黏膜、口底黏膜及舌前 2/3。**（1~8 舌侧）**
7. 颊（长）神经阻滞麻醉 (block anesthesia of buccal nerve)：
	（1）下牙槽神经阻滞麻醉触及骨壁后退回 **2 cm**，回抽无血注射 ==0.5～1 mL==。
	（2）麻醉区域：同侧下颌第二前磨牙及磨牙颊侧牙龈、黏膜、颊部黏膜、颊肌和皮肤。（5~8 颊 ce）
8. 下牙槽、舌、颊神经一次阻滞麻醉（下颌支内侧隆突注射法）(injection on internal ramus prominence)

| 项目   | 下牙槽、舌、颊神经一次阻滞麻醉                |
| ---- | ------------------------------ |
| 注射标志 | 下颌支==内侧隆突==位于下颌小舌的前上方          |
| 进针点  | 翼下颌皱襞外侧，相当于上颌第三磨牙𬌗面下 0.5 cm 处 |
| 进针方向 | 注射器对侧口角，针体与患侧颊黏膜面接近垂直刺入        |
| 进针深度 | ==2 cm 触及骨壁==                  |
| 剂量   | ==1.5～2 ml==，退回少许，再注入 0.5 ml   |

### 三、局部并发症

#### （一）晕厥（syncope）

1. **定义**：由于一时性中枢缺血所致的突发性、暂时性的意识丧失。
2. 病因：恐惧、饥饿、疲劳及全身健康较差等内在因素，以及疼痛、体位不良等因素引起。
3. 临床表现：头晕、胸闷、面色苍白、全身冷汗、四肢厥冷无力、**脉搏快而弱**、恶心、呼吸困难；重者心率减慢，血压急剧下降，甚至有短暂的意识丧失。
4. 预防：做好术前检查及思想工作。消除紧张情绪，==避免空腹==。
5. 治疗：
	（1）一旦发生晕厥，应立即==停止注射==，迅速==放平坐椅==，置病员于==头低位==。
	（2）松解衣领，保持呼吸通畅，芳香氨酒精或氨水==刺激呼吸==。
	（3）针刺人中穴。
	（4）氧气吸入和静脉注射高渗葡萄糖液。

#### （二）、过敏反应 (allergic reaction)
1. **定义**：由细胞和（或）体液介导的，对不同浓度的抗原所产生的的反应。
2. 临床表现分类
   （1）==延迟反应==：①血管神经性水肿（常见）；②荨麻疹、药疹、哮喘和过敏性紫癜（偶见）。
   （2）==即刻反应==：极少量药后，立即发生极严重的类似中毒症状，病人突然惊厥、昏迷、呼吸、心搏骤停而死亡。
3. 预防：术前详细询问有无过敏史
4. 治疗：
	（1）对轻症——脱敏药物如钙剂、异丙嗪、可的松类激素肌注或静注及吸氧。
	（2）对严重过敏反应出现抽搐或惊厥史，应迅速静注安定（地西洋）10～20 mg 或分次静注 2.5% 硫喷妥钠 3～5 ml/次，直至惊厥停止。

#### （三）中毒（过量反应）(over dose reaction)

1. **定义**：当单位时间内进入血循环的局麻药速度超过分解速度时血内浓度升高，达到一定的浓度时就会出现中毒症状。
2. 临床表现：**最早期表现为口周麻木**
   （1）兴奋型：表现为烦躁不安、多话、颤抖、恶心、呕吐、气急、多汗、**血压上升**，严重者出现全身抽搐、缺氧、发绀。
   （2）抑制型：迅速出现脉搏细弱、**血压下降**、神志不清，随即呼吸、心跳停止。
3. 预防：①坚持回抽无血；②不超过最大用量。
4. 治疗：
	（1）**轻**微病人——平卧解衣扣，保持呼吸通畅，待麻药自行分解；
	（2）**重**症病人——给氧、补液、抗惊厥、激素及升压药等抢救措施。

#### （四）注射区疼痛、水肿
1. 原因：
   （1）注射药物变质或混有杂质或未配等渗溶液。
   （2）注射针头钝或弯曲。
   （3）未执行无菌操作。
2. 防治原则：
   （1）预防：①检查药品和器械。②注意消毒隔离。③避免同部位反复操作。
	（2）治疗：①热敷理疗、封闭。②消炎、止痛。

#### （五）、血肿
1. 原因：注射针刺破血管所致，常见上牙槽**后**神经阻滞麻醉刺破翼静脉丛，**眶下**神经阻滞麻醉。
2. 症状：局部迅速肿胀，无疼痛。色泽的改变：紫红色，逐渐变浅呈黄绿色。
3. 预防
   （1）注射针尖不能粗钝及有倒钩。
   （2）注射时不要反复穿刺以免增加穿破血管的机会。
4. 治疗
   （1）出现血肿，可立即压迫止血，并予局部冷敷。
   （2）出血停止后，48 h 后用热敷。
   （3）酌情给予抗生素及止血药物。

#### （六）感染
1. 原因：注射针被污染、局部或麻药消毒不严或注射针穿过感染灶。（与下牙槽神经阻滞麻醉引起的翼下颌间隙感染鉴别）
2. 症状：麻醉后 **1~5 天**出现红肿热痛。
3. 预防：
	（1）注射器械及注射区严格消毒；
	（2）注射时防止注射针的污染；
	（3）避免穿过或直接在炎症区注射。
4. 治疗：抗炎治疗。

#### （七）暂时性面瘫
1. 原因：下牙槽神经口内阻滞麻醉时发生，注射针偏向后内不能触及骨面（角度小）或偏上越过乙状切迹，==注入腮腺麻醉面神经==出现暂时性面瘫。
2. 症状：暂时性面瘫。
3. 治疗：无需特殊处理。

#### （八）暂时性复视或失明
1. 原因：下牙槽神经口内阻滞麻醉时发生，未回抽进入下牙槽动脉，推注的麻药逆行进入==脑膜中动脉或眼动脉==所致。
2. 症状：暂时性复视或失明
3. 治疗：无需特殊处理，坚决回抽预防。

#### （九）暂时性牙关紧闭（trismus）
1. 原因：下牙槽神经口内阻滞麻醉时发生，麻醉药注入==翼内肌==或==咬肌==内，使肌肉失去收缩与舒张的功能所致。
2. 症状：牙关紧闭。
3. 治疗：无需特殊处理。
- 注：暂时性面瘫、暂时性牙关紧闭、暂时性复视或失明均常见于下牙槽神经阻滞麻醉，且均不需要特殊处理。

#### （十）神经损伤
1. 原因：注射针穿刺或撕拉，或注入混有酒精的溶液。
2. 预防：
	（1）注射器械不要有倒钩
	（2）注射时不要反复穿刺
	（3）严格控制药品的污染
3. 治疗：早期给予积极处理（营养神经），促进神经功能的完全恢复。

#### （十一）注射针折断（broken needle）
1. 原因：注射针弹性差、锈蚀、缺乏弹性；病人骤然移动，操作不当
2. 预防：（1）检查注射针质量；（2）注射针保留在体外至少 **1 cm**
3. 治疗：
	（1）未完全进入组织内：持针器去除
	（2）完全进入组织内：X 线定位、手术取出

#### （十二）麻醉后黏膜病变
1. 表现：注射部位出现，多个疱疹性小溃疡；腭部多见，可有疼痛
2. 预防：（1）避免适用含 1：5 万肾上腺素的局麻药物（2）避免注射局部组织过度苍白或注入药液过多。
3. 治疗：（1）局部应用止痛药物 （2）促进局部愈合的药物

#### （十三）颈丛神经阻滞麻醉的并发症
1. 原因：颈深神经阻滞麻醉时，麻药浸润使交感神经麻痹所致。
2. 临床表现：颈交感神经综合征（**霍纳 Horner 征**） #名解
	- 同侧瞳孔缩小、上睑下垂、睑裂变小、结膜充血、面色潮红、耳郭红润、面部皮肤干燥无汗、鼻黏膜充血、鼻塞等。（似醉酒）
3. 治疗：无需处理。
4. 声音嘶哑：喉返神经传导受阻所致。
5. 全脊髓麻醉：误入颈椎椎管蛛网膜下腔所致。

#### （十四）感觉异常（paresthesia）
1. 原因：注射针刺入神经，或注入混油酒精、防腐剂的溶液。
2. 临床表现：感觉异常，部分或完全麻木
3. 治疗：早期给予积极处理，促进神经功能的完全恢复。可以采用针刺、理疗，给予激素（损伤早期）、维生素 B 1 或 B 12 等治疗

## 第二节 口腔颌面外科手术全身麻醉

### 一、常用的全麻方法

#### （一）方法

吸入麻醉、静脉麻醉、基础麻醉、静脉吸入复合麻醉和全凭静脉复合麻醉。

#### （二）全身麻醉的实施准备和诱导

目前全麻的诱导通常选择静脉诱导法。
1. 气管内插管：主要手段之一，口腔颌面外科疾病——经鼻盲探插管。
2. 麻醉维持。
3. 麻醉苏醒和气管拔管：手术完毕前 5～10 min 停止麻醉。
4. 控制性降压麻醉。
5. 低温麻醉。

### 二、全身麻醉的特点及全麻后的处理

口腔颌面外科手术全身麻醉的特点：
1. 麻醉与手术互相干扰。
2. 保持气道通畅比较困难。
3. 小儿与老年患者比例高，出现情况及时处理。
4. 手术失血较多。
5. 麻醉的深度和麻醉恢复期的要求：其深度相当于乙醚吸入麻醉的三期一级。
6. 拔除气管导管的时机很重要，必要时可用药物催醒，待咳嗽、吞咽反射、肌张力恢复后再行拔管。

### 三、镇静与镇痛

#### （一）镇静

1. 镇静的特点：意识存在，服从指令；呼吸、循环变化小；无镇痛作用。
2. 镇静的方法：口服、肌注、静脉。氧化亚氮吸入（含氧不低于 30%）：操作简单、镇静深度易调节、安全。但对气胸、肠梗阻和中耳疾病的患者禁用。

#### （二）镇痛

1. 药物镇痛
   （1）第一阶梯：非甾体抗炎药（非类固醇）。
   （2）第二阶梯：可待因或其他弱阿片类药物。
   （3）第三阶梯：强阿片类镇痛药。
2. 口腔颌面部大手术后或癌性疼痛：阿片类制剂，常作为首选药物。
3. 慢性疼痛尤其是癌症患者的镇痛：宜辅抗抑郁用药（苯二氮䓬类药物）。
4. 中度及重度疼痛：应选用阿片类镇痛药。

注：ICU 只对某些急症或危重病的治疗有意义，即只能帮助病情程度中等或中等偏重的患者，对某些慢性患者晚期、恶性肿瘤晚期、病因不能纠正的濒死患者，ICU 是不收治的。
