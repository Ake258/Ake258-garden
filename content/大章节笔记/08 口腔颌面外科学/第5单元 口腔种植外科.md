# 第五单元 牙种植术

## 第一节 概论
### 一、口腔种植学的发展简史
1. 口腔种植学（oral implantology）:20 世纪 30 年代发展起来的一门独立的新兴分支学科，主要包括种植外科、种植义齿修复、种植材料、种植力学及种植生物学等内容。
2. 口腔种植体（oral implant）又称牙种植体（dental implant）：是指为支持义齿修复的上部结构，用外科手段在上颌或下颌骨内植入人工材料设计的装置。(国际标准化组织.ISO.1984 年)，是起支持、固位作用的植入物。 #名解
3. 现代口腔种植学奠基人：Formiggini
4. 现代口腔种植之父：瑞典 Branemark 教授，提出“骨结合 (osseointegration)”理论

`牙种植是将人工牙（通常指人工牙根）植入牙槽骨内的手术。`
适用于 [国际标准化组织（ISO）1984 年]。
骨内种植体（最常见），种植材料：钛及钛合金。

### 二、口腔种植材料分类

#### （一）分类
1. 按植入部位：
	（1）骨内种植体 (endosseous implant) **最多**
	（2）骨膜下种植体 (subperiosteal implant)
	（3）牙内骨内种植体 (endodontic-endosseous implant)
	（4）黏膜内种植体 (Intramucous membrane implant)
	（5）穿下颌种植体 (transmandibular implant)
	（6）下颌支支架种植体 (ramus frame implant)
2. 按手术方式：一段式和两段式种植体
3. 按不同外形：叶状、螺旋形、桶状、柱状及根形

#### （二）几种不同种植体

| 几种不同种植体                    | 内容                                                        |
| -------------------------- | --------------------------------------------------------- |
| 螺旋型种植体（screw-type implant） | 代表：Branemark 种植系统<br>特点：利用种植体的表面螺纹来增加骨接触面积<br>分类：自攻型、非自攻型 |
| 柱状种植体（cylindrical implant） | 特点：表面多采用各种涂层技术形成粗糙面                                       |

种植体的表面处理（implant surface treatment）  
1. 目的：提高骨结合的速度与质量
2. 方法：
	（1）喷砂酸蚀 (sandblasted large-grit, acid-etched, SLA)
	（2）电化学表面氧化处理 (electrochemical oxide)
	（3）羟基磷灰石涂层 (hydroxyapatite coating)

#### （三）种植材料
目前种植材料应用最广泛：**钛及钛合金**，其优点：
1. 良好的生物相容性。
2. 密度低，机械强度高。
3. 弹性模量较其他医用金属更接近骨组织。
4. 有良好的机械加工性能。

### 三、口腔种植外科手术器械
以纯钛两段式螺旋型牙种植体为例，按种植手术分期予以简单介绍。

#### （一）第一期种植体植入器械
1. 种植机：主机、马达和机头，==快速为≤2000 r/min==、==慢速为≤20 r/min==
2. 钛质种植工具：钛钳、钛镊、连接器、方向指示器等
3. 钻头：球钻、一号裂钻、定向钻、二号裂钻、肩台钻
4. 其他器械：种植体旋入扳手、旋入器、种植体固定扳手等

#### （二）第二期种植体—基台连接术器械
1. 将种植体愈合基台（基台）连接在牙种植体上
2. 器械：牙龈环切刀、骨旋刀、小骨膜剥离器、小骨凿
3. 牙龈厚度测量尺以及各种螺丝扳手等种植体显露及连接基台器械

## 第二节 口腔种植的生物学基础

### 一、种植体与骨组织间的界面
#### （一）骨结合理论
1. 纤维—骨性结合（模拟牙周膜结构）
	（1）定义：种植体与骨组织之间存在着一层非矿化的纤维结缔组织。
	（2）实质：异物反应
	（3）结果：==界面软组织膜使种植体最后感染松动==。
2. **骨结合** (osseointegration)—种植体—==骨界面的正常愈合==
	（1）**理论**：是指牙种植体与具有活性的骨组织产生持久性的骨性接触，界面无纤维介入，并将其定义为“负载的种植体表面与周围发育良好的骨组织之间在结构和功能上的直接结合”。
	（2）骨结合式种植体：负载咬合力的种植体的表面与有活力的骨组织之间存在结构上和功能上直接的联系，种植体与骨组织之间不间隔以任何组织。

#### （二）骨结合三阶段与确认
1. ==第一阶段==：植入后表面被血块包绕，骨髓内生物高分子吸附，形成==适应层==（conformation layer）。
2. ==第二阶段==：术后 1 个月组织破坏与增生同时发生的修复期。（钻骨切削引起的骨损伤多称为松动原因。最容易失败的时期。）
3. ==第三阶段==：术后 3 个月，种植体周围形成胶原纤维，随后形成网状纤维结构，逐步完成骨结合。
4. 种植体骨结合状态的确认：
   （1）临床检查：临床检查种植体==无松动==，叩之==有清音==，==无疼痛==。
   （2）X 线检查：X 线检查种植体与骨组织紧密结合==无透射间隙==。
   （3）动物实验的组织学结果：种植体表面为成骨细胞包绕，界面无结缔组织。

### 二、种植体与软组织（牙龈）界面
#### （一）龈界面
1. 龈界面定义：牙龈软组织与种植体接触形成的界面。
2. **袖口（cuff）**：上皮细胞黏附在种植体表面而形成的生物学封闭
	- 结合方式：﻿ 牙龈上皮细胞分泌细胞外基质在细胞膜和钛氧化膜之间 形成 [[半桥粒]]（hemidesmosome） 
3. **种植体接龈部分的物质表面微形态**，与龈附着也有很大关系
	- 要求：==非常光洁==，若粗糙则易堆积菌斑、附着结石，不利于种植体与牙龈结合，引起牙龈炎症，以至破坏生物封闭状态。
4. 种植体的**生物学宽度**﻿( biological width , BW)：与天然牙类似，种植体周围屏障上皮和结缔组织附着共同构成种植体的生物学宽度，即从膨胀上皮最冠方到牙槽嵴顶之间约 **3~4 mm** 的距离，也称为==生物学屏障==（biological barrier）

### 三、影响种植体骨结合的因素
1. 手术创伤：钻孔时产热不超过于 **47℃**（主要原因）；骨钻转速不超过 **2000r/min** 。
2. 患者自身条件。
3. 种植体材料的生物相容性。
4. 种植体外形设计（表面的粗化处理）。
5. 种植体的应力分布。
6. 种植体的早期负载。

## 第三节 口腔种植外科的应用解剖
### 一、缺牙后牙槽突的改变
缺牙区，尤其是全牙列缺失后，失去咬合力这一生理学刺激，牙槽突均有不同程度的萎缩或吸收。

| 牙列缺失后 | 萎缩方向                          |
| ----- | ----------------------------- |
| 上颌    | 牙槽突长轴唇倾，  牙槽基部位置靠后， 唇侧骨板比腭侧薄； |
| 下颌    | 牙槽突长轴内倾，牙槽基部靠前， ﻿舌侧骨板薄于居侧，    |
| 骨吸收后  | 结果造成下颌牙弓相对==大于==上颌牙弓的==反颌状态== |

### 二、牙槽嵴分类

#### （一）形态学分类
Lekholm 和 Zarb 分类

| 分级  | Lekholm 和 Zarb 分类           |
| --- | --------------------------- |
| A 级 | 大部分牙槽嵴尚存                    |
| B 级 | 中等程度的牙槽嵴吸收                  |
| C 级 | 明显的牙槽嵴吸收，仅基底骨（basal bone）尚存 |
| D 级 | 基底骨已开始吸收                    |
| E 级 | 基底骨已发生重度吸收                  |

#### （二）骨的质量分类
根据骨密质与骨松质的含量比例及骨松质疏密程度，将颌骨质量分为 ==4 级==

| 分级     | 骨的质量分类              | 缺点            |
| ------ | ------------------- | ------------- |
| Ⅰ级     | 颌骨几乎完全由均质的骨密质构成     | 太厚，切削产热过多     |
| ==Ⅱ级== | 厚层的骨密质包绕骨小梁密集排列的骨松质 | 最好            |
| Ⅲ级     | 薄层的骨密质包绕骨小梁密集排列的骨松质 | 好             |
| Ⅳ级     | 薄层的骨密质包绕骨小梁疏松排列的骨松质 | 太薄疏松，植体初期稳定性差 |

## 第四节 口腔种植手术

### 一、适应证
除禁忌证以外均为适应证，可解决以前不能解决的很多问题。如：
1. 部分缺失、邻牙不能做基牙
2. 游离缺失
3. 无牙颌不能用传统义齿修复者

### 二、禁忌证
全身不能耐受、局部有病变。

| 禁忌证 | 禁忌证                  |
| --- | :------------------- |
| 全身  | 不能承受手术者              |
|     | 严重糖尿病                |
|     | 骨疾病如：骨质疏松症、骨软化症、骨硬化症 |
| 局部  | 口腔内有急、慢性炎症者          |
|     | 颌骨内良、恶性肿瘤者           |
|     | 严重习惯性磨牙症             |
|     | 口腔卫生不良者              |
|     | 精神病患者                |

### 三、治疗程序
以==两段两次法==骨水平种植体（bone level implant）为例，通过先后两次手术植入牙种植体及连接其上部结构，最后完成种植义齿修复。

#### （一）第一期手术
种植体固位钉植入缺牙部位的牙槽骨内。术后 7~10 天拆线。

#### （二）第二期手术
1. 时间：一期手术后 **3~4 个月**种植体完成骨结合后。==上颌 4 个月==（松），==下颌 3 个月==（密，所以时间短）
2. 过程：①安装穿龈的愈合基桩（healing abutment）。②==第二期手术后 14~20 天即可取模==制作义齿

#### （三）复诊
种植义齿修复后，==第一年每 3 个月复查一次，以后每年至少复查 2 次==。

### 四、种植体植入术的原则

1. ==手术的微创性==：损伤主要包括机械损伤和热灼伤，制备种植窝时骨床温度不应**超过 47℃**，以保证微创性。
2. ==牙种植体表面无污染==（血渍不是）：①细菌污染；②脂类及异种蛋白污染；③异种金属元素污染。
3. ==牙种植体的早期稳定性==：是界面实现骨结合的基本愈合环境。种植体旋入的扭力 **≥ 35 N·cm** 时视为固位良好，低于此值应采用埋入式植入或延长愈合期。
4. ==种植体愈合无干扰==：指牙种植体在骨愈合过程中不受口腔微生物环境及过早负载不利因素的影响。种植体旋入的扭力 **≥ 35 N·cm**，且术后控制其微动（50~150μm），是进行即刻负重的重要先决条件。
5. 牙种植两次法的基本原则：①牙龈黏膜严密缝合，避免口腔微生物环境及过早咬合力影响。②种植体不应过度高于骨面。③避免承受咬合力而对成骨过程造成的不利影响。

#### （一）受植区的要求

缺牙间隙近远中径至少 6mm，颌龈距离 7mm。

| 项目            | 要求            |
| :------------ | :------------ |
| 种植体唇颊、舌腭侧骨质厚度 | 厚度不能少于 1.5mm  |
| 种植体之间         | 厚度不能少于 3mm    |
| 种植体与天然牙之间     | 厚度不能少于 2mm    |
| 种植体末端距离上颌窦底   | 厚度不能少于 2mm    |
| 种植体末端距离下颌管上缘  | 厚度不能少于 2mm    |
| 一般种植体长度       | 厚度不应少于 8~10mm |

### 五、术前准备
1. 全身：患者全身情况
2. 局部：
	（1）颌骨形态、咬合关系、颌间距离等
	（2）X 线片——颌骨大小、密质骨及松质骨比例、上颌窦有无炎症、窦底位置、颈孔及下颌管位置等。
	（3）植骨区黏膜软组织情况
	（4）在石膏模型上设计确定种植体植入的方向、位置、数目等。余留牙应彻底检查、治疗并常规作洁治。

### 六、麻醉及体位
牙种植术主要采用口内局部浸润麻醉。
1. ==下颌==：下牙槽神经、舌神经及颊神经阻滞麻醉
2. ==上颌==：上牙槽前、中或后神经、腭大及鼻腭神经浸润麻醉
- 全麻插管：骨缺损骼骨移植、游离腓骨移植种植

### 七、常用牙种植手术
#### （一）全口缺牙下颌牙种植术
牙列缺损患者由于牙槽嵴萎缩常导致下颌全口义齿固位差，因此下颌无牙颌患者多选择种植义齿修复。
1. 术前设计：下颌
	（1）尖牙区各植入 ==1== 枚种植体，以==杆卡式==修复。
	（2）颏孔间植入 ==4~6== 枚种植体，以==杆卡式或固定式==修复。
	（3） 除颏孔间种植外， 两侧前磨牙及磨牙区各种植 ==1~2== 枚种植体，以==可摘式或固定式==修复。
	- 上颌：前牙及尖牙区种植，磨牙区骨质少，常需==植骨或上颌窦底==提升才能种植。
2. 手术步骤
	（1）一期手术：种植体固位钉 (fixture) 植入术
	（2）二期手术：种植体—基台 (abutment) 连接术
	（3）第一期手术后 **3~4 个月**待种植体与颌骨完成骨结合后，再行二期手术，安装基台。
3. 注意事项
	（1）切开下颌黏膜时注意**勿**损伤颏神经，尤其是下颌牙槽嵴严重萎缩者，
	（2）上颌前牙种植应特别**注意**牙槽嵴唇 - 腭厚度，必要时植骨。
	（3）下颌种植体的固位不宜穿出下颌下缘过多。上颌植体可穿出窦底及鼻底，但绝对不能穿破黏膜，否则会因感染而失败。
	（4）种植的部位和方向要考虑修复后义齿的咬合、覆盖关系以及能否获得共同就位道；
	（5）制备种植窝高速钻要低于 **2000 r/min**，并用等渗生理盐水不断冲洗降温，以使局部产热**不超过 47°C**，
	（6）基台与种植体固位钉之间不能有任何物质间隔；各组件连接之前，必须去除所有软硬组织干扰，以保证金属配件紧密旋紧。

#### （二）单个牙种植术
1. 切口：距牙槽嵴顶 3 mm，唇腭侧切口
2. 种植植入部位及方向：
	（1）部位：种植入口应==稍偏腭侧==。
	（2）方向：与邻牙长轴==基本平行==，牙槽骨唇腭侧之正中。
3. 基台安装：基台穿龈不能超过 ==2 mm==，以便恢复良好的龈边缘形态。

## 第五节 种植牙区骨量不足的处理

### 一、引导骨再生术（GBR）
引导骨再生（guided bone regeneration , GBR）术基于引导组织再生（guided tissue regeneration , GTR）技术发展而来。
其原理是根据各类组织细胞迁移速度不同的特点，将屏障膜置于软组织和骨缺损之间建立**生物屏障**，创造一个相对封闭的组织环境，阻止结缔组织细胞和上皮细胞进入骨缺损区，允许有潜在生长能力、迁移速度较慢的==前体成骨细胞==优先进入骨缺损区，==优势生长==，同时保护血凝块，缓解压力，实现缺损区的骨再生。
- 屏障膜的分类
	1. 不可吸收（生物不降解）：代表为膨体聚四氟乙烯膜 (expanded polytetrafluoroethylene，ePTFE，又称 Gore-Tex 膜)
	2. 可吸收（生物降解）：代表为：聚乳酸膜、聚乙烯酯膜、共聚物膜及胶原膜，以及 Bio-Gide 等
![[图4-3 种植骨缺损植入引导骨再生膜.png|573]]

### 二、下牙槽神经解剖移位

1. 磨牙缺失过久必然造成牙槽嵴吸收，下颌管以上骨高度不足 **8 mm** 者，可行颊侧开窗解剖游离下牙槽神经血管并向颊侧移位，再从嵴上植入种植体；
2. 若骨吸收严重，颏孔、下颌管仅位于骨表面时，可行骨表面上开窗将该神经血管解剖移位进行种植。

### 三、上颌窦底提升
上颌磨牙区，尤其是上颌窦底到牙槽嵴的距离过小，缺乏足够的骨组织支持时，一般采用上颌窦底提升术（sinus floor elevation）来解决骨量不足的问题。
1. 侧壁开窗法（lateral window technique）
2. 骨凿冲顶法（osteotome technique）

### 四、牵张（引）成骨术
牵张（引）成骨术（distraction osteogenesis , DO）是在肢体长骨牵张成骨技术的基础上发展起来的新技术，已被引入牙槽外壳。除常规牵张器外，还有以种植体作为牵张器的设计。
其治疗过程可分为四个阶段，即骨切开、延迟期、牵张期与固定器。具体要求见十四章：颌骨牵张成骨

## 第七节 即刻种植
即刻种植（immediate）是指在牙拔除后立即进行种植体植入的方法。

### 一、即刻种植的适应证
1. 种植位点无急性根尖周病和牙周病
2. 无法保留的患牙根尖区有根尖周肉芽肿但范围局限
3. 牙槽窝根方至少有 3~5 mm 骨量，保证种植体能获得足够的初期稳定性
4. 尤其适用于单根牙拔除后

### 二、即刻种植的外科程序
1. 切口
2. 微创拔牙
3. 种植窝制备及种植体植入
4. 骨缺损的处理
5. 关闭创口

## 第九节 种植手术并发症

### 一、术中并发症
#### （一）术中出血
1. 原因：①翻瓣时损伤血管②高血压③长期使用抗凝药物④备孔或去骨时伤及血管
2. 处理：止血—对因

#### （二）窦腔黏膜穿孔
1. 原因：骨量不足
2. 处理：应及时去除种植

#### （三）神经损伤
1. 原因：术中损伤下牙槽神经（最常见）、颏神经、下颌切牙神经或舌神经所致
2. 处理：前者多可恢复。后者应去除种植体，避开神经重新植入

#### （四）邻牙损伤及侧壁穿孔
种植体与邻牙及侧壁必须保持至少 1.5 mm 间距，如备孔时方向偏斜，有可能造成邻牙损伤或侧壁穿孔。

### 二、术后并发症
#### （一）术后急性感染
主要表现为：种植区肿胀、疼痛、创口红肿、有分泌物渗出，后期可有脓肿或窦道形成。严重时可伴张口受限和头疼，也可能伴有发热和区域淋巴结肿大。

#### （二）种植体骨结合不良
指种植体在植入后至修复前，种植体和骨组织之间的骨结合不完整，或没有骨结合，只有纤维结合，造成种植体松动或脱离。

#### （三）术后出血及皮下瘀斑
种植手术后 24 h 若仍有持续性的活动性出血或明显的血块形成，则属于术后出血（postop-erative bleeding），要及时止血。
皮下瘀斑（ecchymosis）一般在手术后 1~2 天内出现。

#### （四）创口裂开
缝合不当、患者年龄过大、附着龈缺乏、创口感染、过渡性义齿的压迫、术区有瘢痕组织以及吸引与酗酒等不良生活习惯等都是引起创口裂开的风险因素。

| 并发症 #简答   | 原因                                                   |
| --------- | ---------------------------------------------------- |
| 感染        | 未严格无菌操作<br>处理：抗炎                                     |
| 牙龈炎       | 口腔卫生不良或清洁方法不当所致                                      |
| 牙龈增生      | 由于==基桩穿龈过少或与桥架连接不良==所致                               |
| 进行性边缘性骨吸收 | 多发生在种植体颈部的骨组织，与牙龈炎、种植体周围炎、种植体应力过于集中以及种植体机械折断长时间未纠正有关 |
| 种植体创伤     | 种植体义齿被意外撞击                                           |
| 种植体机械折断   | 主要因机械因素或应力分布不合理所致                                    |

## 第十节 种植义齿成功标准
#### （一）1995 年《中华口腔医学》杂志社在珠海制定的种植成功标准⭐⭐

| 种植成功标准            | 种植成功标准                                                 |
| ----------------- | ------------------------------------------------------ |
| ①功能好              | ②无麻木，疼痛等不适                                             |
| ③自我感觉良好           | ④种植体周围==X 线无透射区==；**横行骨吸收不超过 1/3**，种植体不松动（国外是**垂直**方向） |
| ⑤**龈炎可控制（可以有龈炎）** | ⑥无与种植体相关的感染                                            |
| ⑦对邻牙支持组织无损害       | ⑧美观                                                    |
| ⑨咀嚼效率**大于 70%**   | ⑩符合上述要求者**5 年成功率应达到 85% 以上；10 年 80% 以上**               |

#### （二）1986 年瑞典 Albrektsson 和 Zarb 等提出的口腔种植成功标准⭐

1. 种植体无动度  
2. X 线片显示种植体周围无透射区  
3. 种植体**功能负载 1 年后**，**垂直**方向骨吸收 **小于 0.2 mm**/年  
4. 种植体无持续性或不可逆的症状，如疼痛、感染、麻木、坏死、感觉异常及下颌管损伤  
5. 达上述要求者，**5 年**成功率 **85%** 以上；**10 年**成功率 **80%** 以上为最低标准

#### （三）1978 年美国国立卫生研究院 (NTH) 召开的研讨会提出口腔种植成功的评价标准
1. 种植体在任何方向上的动度小于 1 mm（❌）
2. 在 X 线片上，种植体周围无透射区
3. 垂直方向的骨吸收不超过种植体的 1/3
4. 允许有可治愈的牙龈炎；无症状，无感染，无邻牙损伤，无感觉异常及麻木，无下颌管、上颌窦及鼻底组织的损伤
5. **5 年**成功率应达到 **75%**
