# 第一单元 口腔黏膜感染性疾病

1. 病毒：单纯疱疹、带状疱疹、手足口病
2. 细菌：细菌性口炎
3. 真菌：口腔念珠菌病

## 第一节 口腔单纯疱疹

### 一、定义

单纯疱疹（herpes simplex）是由**单纯疱疹病毒**（**HSV**,herpes simplex virus）所致的皮肤黏膜病。
临床上以出现**簇集性小水疱为特征，有自限性，易复发。** 单纯疱疹病毒在人群中分布广泛，感染率高，可导致多种疾病，如龈口炎、角膜结膜炎、脑炎、生殖道感染以及新生儿感染等。

### 二、临床表现

#### （一）口腔单纯疱疹临床表现

**簇集性小水疱为特征，自限性，易复发。**

#### （二）病毒分型

HSV-1（==腰部以上==，涉及疾病：==唇癌）==。
HSV-2（腰部以==下==，涉及疾病：==宫颈癌==）。
口腔单纯疱疹分型如下。

#### （三）临床表现

| 分型                          | 特点                                                                                             |
| :-------------------------- | :--------------------------------------------------------------------------------------------- |
| **原发性疱疹性口炎**（又称 `急性疱疹性龈口炎`） | 好发==6 岁以下，6 月～2 岁婴幼儿更为多见==。（6 个月前有来自母体的被动免疫）。也可见于成人。                                           |
| **复发性疱疹性口炎**（又称 `复发性唇疱疹`）   | ==口唇或接近口唇处好发==（==30%～50%== 可复发），总是以疱开始，总是在原发作过的位置，愈后不留瘢痕，可能会有色素沉着。<br> 诱发因素：阳光、局部机械损伤，轻度发热感冒、情绪 |

发病过程：==前驱期、水疱期、糜烂期、愈合期==（结痂期）
1. 前驱期
	潜伏期为 4~7 天后，出现发热、头痛、淋巴结肿大等。==患儿流唾液，拒食、烦躁不安==。经过 1~2 天后，口腔黏膜广泛充血水肿。
2. 水疱期
	**任何部位**可发生**成簇**似针头小水疱
3. 糜烂期
	**成簇**水疱溃破后可形成大面积==糜烂，上覆黄色假膜==
4. 愈合期
	整个病程约为 **7~10 天**，病后有抗体，但**不能防止复发**。

### 三、诊断及鉴别诊断

（1）诊断　
	根据临床表现和实验室诊断为最终确诊。
	病毒的分离培养鉴定是**确诊金标准**（特异）
（2）鉴别诊断

|        疾病名称         | 病损特点                                                                                       |
| :-----------------: | :----------------------------------------------------------------------------------------- |
|       疱疹性龈口炎        | **6 个月～2 岁婴幼儿**，==急性==发作，全身反应较==重==<br>①**成簇小水疱**，疱破后成为==大片表浅溃疡==  ②**口腔粘膜各处** ③**可伴**皮肤损害 |
| 口炎型口疮<br>（疱疹样阿弗他溃疡） | **成人**，==反复发作==，全身发应较==轻==。<br>①**散在**==小溃疡==（满天星），无发疱史 ②损害多发于口腔的**无角化黏膜** ③**无**皮肤损害      |
|      三叉神经带状疱疹       | 夏秋季，==水痘 - 带状疱疹病毒（VZV）==，沿**三叉神经**的分支排列成带状，==不超过中线==。**明显神经痛**。                            |
|        手足口病         | 夏秋季，==柯萨奇 A16 或肠道病毒 EV71 病毒==。**3 岁儿童**为主要患病人群                                             |
|       疱疹性咽颊炎        | 夏季，==柯萨奇 A4 型病毒==，病损的分布只限于口腔后部，如软腭、悬雍垂、扁桃体处，为丛集成簇的小水疱，不久破溃成溃疡，病程大约 7 天。                    |
|        多形性红斑        | 诱发因素包括感染、药物。累及皮肤和黏膜，以**靶形**或**虹膜状红斑**为典型皮损，最常见的病变为==大面积糜烂==。                               |

## 第二节 带状疱疹

### 一、定义

带状疱疹（herpes zoster）是由**水痘—带状疱疹病毒**（**VZV**,herpes varicella-zoster virus）所引起的，以沿单侧周围神经分布的簇集性小水疱为特征，常伴有明显神经痛。

### 二、临床表现

（1）本病==夏秋季==的发病率较高。
（2）最常见胸腹或腰部带状疱疹 70％，三叉 20％，不留痕迹。60 岁以上老人三叉＞腰腹。
（3）沿单侧周围神经分布的簇集性小水疱，不超过中线，常伴有**明显的神经痛**。
（4）损害愈合后在一段时间内仍有**疼痛**，愈合后不再复发。疼痛超过**一个月以上**为==带状疱疹后遗神经痛==。
（5）Ramsay-Hunt(RH) 综合征：==面瘫 + 耳痛 + 外耳道疱疹（侵犯膝状神经节）==。

### 三、诊断

特征的单侧性皮肤 - 黏膜疱疹，沿神经支分布及剧烈的疼痛。

### 四、治疗

1. 抗病毒药物；
2. **止痛（卡马西平）**；
3. 营养神经药物（维生素 B₁或维生素 B₁₂）
4. 糖皮质激素
5. 局部治疗（漱口水、抗病毒软膏、紫外线和红外线）
6. 免疫调节：转移因子、胸腺肽、人免疫球蛋白

## 第三节 手足口病（HMFD）

### 一、病因

病毒：==柯萨奇 A16（Cox A16）==或==肠道病毒 71 型（EV-A71）==。

### 二、临床表现

（1）发病季节　夏秋季。
（2）发病人群　**3 岁以下幼儿**是主要患者。
（3）临床特点　==离心性分布==。**口腔黏膜、手掌、足底**出现散在水疱、丘疹与斑疹，数量不等。
潜伏期约 2~10 天，平均 3~5 天。多数无浅龋症状二突然发病。常有 1~3 天的发热、咳嗽、流涕、食欲缺乏等症状。

## 第四节 球菌性口炎 (coccigenic stomatitis)

### 一、定义

球菌性口炎是==急性感染性口炎==的一种，临床上以形成==黄褐色或灰白色==**假膜损害**为特征，又叫做 `膜性口炎`。
主要致病菌：==金黄色葡萄球菌、草绿色链球菌、溶血性链球菌、肺炎双球菌==。

### 二、临床表现

灰白色或黄褐色假膜，**较厚**==微突出粘膜表面，致密而光滑==。
擦去假膜，可见糜烂面。周围粘膜充血水肿。==患者唾液增多，疼痛明显==，有炎性口臭。

### 三、诊断

==涂片及细菌培养==可明确诊断。

### 四、治疗
（1）抗菌药物
（2）补充维生素（维生素 B₁、维生素 B₂、维生素 C）
（3）中药治疗（清热解毒）
（4）局部治疗（漱口水、西地碘片）

## 第五节 口腔念珠菌病 (candidosis,candidasis)

### 一、定义

口腔念珠菌病 (oral candidosis,oral candidasis) 是由念珠菌属感染所引起的口腔黏膜疾病，是人类最常见的口腔**真菌感染**。

### 二、病因

（1）致病菌　**白色念珠菌**（**最主要**）和热带念珠菌（致病力最强），**机会性感染**。
（2）特点　白念是单细胞酵母样**真菌**，==不耐热，喜酸恶碱==。
（3）存在形式　以芽生孢子（==寄生==）和假菌丝（==致病==）形式存在。

### 三、临床表现

#### 分型
（1）**念珠菌性口炎**
	①急性假膜型（鹅口疮）：又称 `鹅口疮` 或 `雪口病`，任何年龄，新生儿最多见，损害区黏膜充血，有散在的**色白如雪的柔软小斑点，帽针头大小**，可融合成白色丝绒状斑片。用力**可擦去**。
	②急性红斑型（萎缩型）：==广谱抗生素、激素长期应用、HIV 感染者==。舌黏膜多见外形==弥散的红斑==，==舌背乳头萎缩==。
	③慢性红斑型（萎缩型）：又称 `义齿性口炎`。多上颌义齿承托区黏膜==弥漫性红斑==，==亮红色水肿==。
	④慢性增殖型（肥厚型）：又称 `念珠菌性白斑`。颊黏膜病损，对称位于==口角内侧三角区==，呈==结节状或颗粒状增生==。
（2）==念珠菌性口角炎==：特征是常为==双侧罹患==，口角区以==湿白糜烂==为特征的真菌性口角炎。
（3）==念珠菌性唇炎==：多发于==高龄==（50 岁以上）患者，一般发生于下唇。
（4）==慢性黏膜皮肤念珠菌病==（CMCC）　自身免疫调节基因缺陷相关疾病，涉及口腔黏膜、皮肤及指（趾）甲。

### 三、诊断

（1）**涂片法**：①直接镜检：10%KOH 氢氧化钾、、芽生孢子、假菌丝；②革兰染色 阳性 ；③PAS 染色：**红芽孢、蓝菌丝**；
（2）组织病理：**微小脓肿**
（3）其他：①分离培养；②免疫、基因诊断

### 四、治疗

| 项目            | 内容                                                        |
| :------------ | :-------------------------------------------------------- |
| 局部药物治疗 **为主** | ①**2%～4% 碳酸氢钠溶液**②氯己定 0.2% 的溶液或 1% 凝胶③西地碘（华素片）④**制霉菌素**⑤咪康唑 |
| 全身抗真菌药物治疗     | **氟康唑**（==临床应用最广，治疗首选==），==伊曲康唑==（XX 康唑）口服治疗浅表感染优于酮康唑。    |
| 支持治疗          | 加强营养，增强机体免疫力、注射转移因子，胸腺肽                                   |
| 手术治疗          | 轻、中度上皮异常增生                                                |

## 第六节 口腔结核

### 一、定义

口腔结核是由**结核分枝杆菌**侵犯黏膜引起的慢性感染。
由于结核分枝杆菌的数量、毒力及机体抵抗力的差异，可呈现不同的临床表现。
口腔软组织的结核病损包括：口腔黏膜==结核初疮==、口腔黏膜==结核性溃疡==、口腔==寻常狼疮==。

### 二、临床表现

#### 分型
（1）结核初创（原发性综合征）：临床少见，也见于儿童，少见于成人。结核杆菌首先入侵的部位，入侵处出现小结可发展为顽固性溃疡。发送于口腔典型损害，多见于口咽部或舌部。无痛、局部淋巴结肿大。
（2）**结核性溃疡**：常见继发性结核损害是结核性溃疡，口腔中**最常见**，可在任何部位发生，多见于舌。边界清楚、暗红色的==桑葚状肉芽肿==。==鼠噬状、潜掘状==边缘。
（3）寻常狼疮：临床上少见，为结核菌素试验阳性这再次感染后产生的皮肤结核病。无结核病灶且免疫功能好的青少年。绿豆大小结节、玻片压诊中央苹果酱色。

### 三、鉴别诊断
确诊：组织病理学检查
（1）创伤性溃疡：创伤因子
（2）恶性肿瘤：基底硬结、边缘坚硬
（3）梅毒：梅毒血清试验
（4）深部真菌感染：真菌感染

### 四、治疗
（1）抗结核治疗：异烟肼、利福平、乙胺丁醇、吡嗪酰胺（一线口服）、链霉素（一线注射）、卡那霉素、阿米卡星（二线注射）
（2）对症抗感染治疗
