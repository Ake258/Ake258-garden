# 第五单元 口腔斑纹类疾病

**口腔白斑病、口腔扁平苔藓**、口腔红斑、盘状红斑狼疮。比较 OLP 和 OLK 的临床表现。

## 第一节 口腔白斑病 OLK

### 一、定义

**口腔白斑病（OLK）** 是指口腔**黏膜**上有白色斑块或斑片，不能以临床和组织病理学的方法诊断为其他任何疾病者。
白斑和红斑归入**癌前病变**或==潜在恶性疾病==（potentially malignant disorders，**PMD**）。
临床上可将白斑分为==临时性诊断==和==肯定性诊断==两个阶段：
1. 临时性诊断：发现白色的黏膜斑块，又不能诊断为其他疾病时。
2. 肯定性诊断：去除某些局部因素后，经 ==1～3 个月==（2～4 周，北医版教材）的观察损害仍持续存在。

### 二、病因

口腔白斑病的发病与局部因素的长期刺激以及某些全身因素有关，目前仍有相当数量的口腔白斑病病因不明。
1. 吸烟、饮酒、嚼槟榔等==理化刺激因素==。
	（1）烟草是口腔白斑病发病的重要因素，其由高到低的顺序是：`吸旱烟>吸纸烟>吸水烟`
	（2）乙醇是发生口腔白斑病的独立危险因素，与酒的类型或饮酒方式无关。
	（3）食用过烫或酸辣食物、嚼槟榔等局部理化刺激
2. ==念珠菌感染==。（白色念珠菌可能是白斑发生的一个重要致病因素或是其中的一种合并因素。）
3. ==人乳头瘤病毒==（HPV）感染。（有待进一步研究）
4. ==全身因素==。（包括微量元素、微循环改变、遗传易感性、脂溶性维生素（A）缺乏等。）

### 四、诊断及鉴别诊断
口腔白斑病的诊断需根据**临床表现**和**病理表现**作出综合性判断才能完成
其**诊断的确定性 (certainty，C）** 可分为以下四个：

| 确定性分级 (C) |     诊断     | 诊断学意义                                                                                                                      |
| :-------: | :--------: | -------------------------------------------------------------------------------------------------------------------------- |
|  **C 1**  |  暂时性临床诊断   | 根据临床初诊检查证据，排除其他可定义的疾病或损害；                                                                                                  |
|  **C 2**  |  肯定性临床诊断   | 去除可能的致病因子 2~4 周后损害无改善；                                                                                                     |
|  **C 3**  | 切取组织病理证实诊断 | 在 C 2 基础上，结合切取组织病理检查未发现其他可定义病损，现为口腔白斑病的损害特征；                                                                               |
|  **C 4**  | 切除组织病理证实诊断 | 外科切除所有临床可见损害，并通过组织病理检查作出的诊断。                                                                                               |
|   诊断记录    |    应书写为    | 病损部位 + 诊断（C 因子等级）。例如，左颊白色病损，临床诊断为口腔白斑病，切取组织病理检查结果为口腔白斑病，伴轻度异常增生，诊断书写为：左颊口腔白斑病伴轻度异常增生（C 3）。这种标准化的记录，有利于不同诊断单位之间信息的交流与资料的共享。 |

### 五、癌前倾向问题

增殖性疣状白斑癌变率较高。**病理检查**有无异常增生及异常增生程度对预测癌变最有价值，是目前预测口腔白斑病癌变风险重要指标。
口腔白斑病患者伴有以下情况者**癌变风险较高**，应严密随访观察，必要时可进行多次组织活检。

| 分类  | 危险因素                                          |
| :-: | --------------------------------------------- |
| 病理  | 伴有上皮异常增生者，程度越重者越易恶变。                          |
| 类型  | 非均质型：疣状、颗粒型、溃疡或糜烂型及伴有念珠菌感染、HPV 感染者易恶变。        |
| 部位  | 白斑位于舌缘、舌腹、口底的 U 形区、口角内侧三角形区、软腭复合体区域者易恶变。      |
| 时间  | 病程越长者越易恶变                                     |
| 吸烟  | 不吸烟患者                                         |
| 性别  | 女性，特别是不吸烟的年轻女性患者。女性和不吸烟者口腔白斑病癌变风险升高的具体原因尚不明确。 |
| 面积  | 白斑病损面积大于 200 mm²的患者易恶变。                       |

### 六、治疗

目前尚无根治的方法，管理的目标是**缓解症状、监测和预防癌变**。
主要的管理措施包括卫生宣教、去除刺激因素、药物治疗、手术治疗、物理治疗、中医中药治疗和定期随访。
1. **首先**去除致病因素。对于小面积的病损可采用手术切除、激光、冷冻。术后必须复查。
2. 保守治疗白斑 - 维生素 A 及其衍生物。
3. 对伴白色念珠菌感染的病损配合抗真菌治疗。
4. 白斑区发现溃疡或基底变硬、表面增厚显著时或已证明具有癌前改变的损害，应及早手术切除。

## 第二节 口腔扁平苔藓 OLP

### 一、定义
口腔扁平苔藓（OLP）是一种皮肤 - 黏膜慢性炎症性疾病。属于**癌前状态**。中年女性患者较多。
1. 口腔扁平苔藓 (oral lichen planus，OLP) 是一种常见==口腔黏膜慢性炎性疾病==，患病率为 0.5%~2%，是口腔黏膜病中仅次于复发性阿弗他溃疡的常见疾病。
2. 该病好发于中年，**女性多于男性**。
3. 多数患者有==疼痛、粗糙不适==等临床症状。皮肤及黏膜可单独或同时发病。约 15% 的口腔扁平苔藓患者伴有皮肤病损；皮肤病损与口腔黏膜病损在临床表现上不同，但其病理表现非常相似。
4. 因口腔扁平苔藓长期糜烂病损有恶变倾象，恶变率为 0.4%～12.5%。WHO 将其列入口腔黏膜潜在恶性疾患 (oralpotential malignant disease）的范畴。

### 二、病因
OLP 的病因和发病机制目前尚不明确，多种致病因素有关，其中细胞介导的局部免疫应答紊乱在 OLP 的发生发展中具有重要作用。
（1）精神因素　50% 有精神创伤史。
（2）感染因素　病毒感染。
（3）免疫因素　上皮下固有层内出现 T 淋巴细胞为主的淋巴细胞浸润带。
（4）内分泌因素 内分泌失调
（5）其他：微循环因素、系统病、遗传因素、微量元素缺乏等。

### 四、诊断
组织病理学检查：上皮下疱、胶样小体。基底细胞液化变性。固有层淋巴细胞浸润带。
直接免疫荧光法：基底膜区网状荧光，主要为 IgM。

一般根据病史及典型的**口腔黏膜白色损害**即可作出临床诊断，==典型的皮肤或指（趾）甲损害==可作为诊断依据之一。
临床上常结合活体组织检查术、必要时辅以免疫病理等实验室检查进行确诊，这也有助于鉴别其他白色病变并排除上皮异常增生或恶
性病变。

### 六、治疗
1. 心理治疗　对病损区无充血、糜烂，患者无明显自觉症状者，可在身心调节的情况下观察。一些患者可自愈。同时注意调节全身状况，如睡眠、月经状况、消化道情况，纠正高黏血症等。
2. 局部治疗　
	（1）去除局部刺激因素，消除感染性炎症。
	（2）局部药物治疗：①糖皮质激素：病损区基底部注射对糜烂溃疡型有较好疗效。②维 A 酸：对于病损角化程度高的患者适用。可以避免系统使用的副作用，肝脏功能损害、唇炎、致畸性。③抗真菌药物：对迁延不愈的 OLP，应注意有白色念珠菌感染可能
3. 全身治疗　
	（1）糖皮质激素：对急性大面积或多灶糜烂型 OLP，可慎重考虑采用小剂量、短疗程方案。成人可选用口服泼尼松 20~30 mg/d，服用 1~3 周。
	（2）免疫抑制剂：羟氯喹，不良反应较多，孕妇忌用。
	（3）免疫增强剂：根据患者免疫状况选用，临床常用的有胸腺肽肠溶片和转移因子等。
	（4）抗氧化剂：口服β- 胡萝卜素、番茄红素
	（5）中成药
	（6）其他：白芍总苷，具有抗炎和免疫调节功能

## 第三节 口腔红斑 OE

### 一、定义
1. 口腔红斑病 (oral erythroplakia) 又称 `增殖性红斑`(erythroplasia of Oueyrat)、`红色增殖性病变`(erythroplastic lesion）等是指口腔黏膜上鲜红色斑片，似**天鹅绒样**，边界清晰。在临床和病理上==不能诊断为其他疾病者==。
2. 红斑属于==潜在恶性疾患==。国内文献及教科书曾译为赤斑，用于与炎症性红斑相区别。本病由奎来特（Queyrat）于 1911 年提出，故也称为 `奎来特红斑`。
3. 口腔红斑病比口腔白斑病少见。据文献报道，口腔红斑的患病率在 0.02%~0.1%，且没有明显的性别差异，红斑病一般没有症状，但部分患者会伴有进食时的烧灼感。

### 二、病因

口腔红斑**病因不明**。目前研究认为口腔红斑的发生与烟酒的摄入以及在此过程中发生的遗传事件有关。
原癌基因激活，抑癌基因损伤——削弱了基因组对细胞正常分裂的监控能力——上皮细胞恶变。

### 三、临床表现

口腔红斑多见于中年患者，男性略多于女性。发病部位以舌缘部最多见龈、龈颊沟、口底及舌腹、腭部次之。临床上分为三种类型：

|                 类型                  | 临床表现                                                                                                           |
| :---------------------------------: | :------------------------------------------------------------------------------------------------------------- |
|  均质型红斑<br>homogenous erythroplakia  | 病变较软，**天鹅绒样**鲜红色表面，光滑、发亮。状似 " 上皮缺失 "。病损==边缘清楚==，约为 0.5~2cm 大小，平伏或微隆起。<br>红斑区内有时也可看到外观正常的黏膜。                       |
| 间杂型红斑<br>interspersed erythroplakia | 红斑病损区内有散在的白色斑点，**红白相间**，有时与扁平苔藓不易区分。                                                                           |
|   颗粒型红斑<br>granular erythroplakia   | 在红斑病损区内有颗粒样微小的结节似**桑葚状**或颗粒肉芽状，稍高于黏膜表面，微小结节为红色或白色。<br>有时其外周亦可见散在的点状或斑块状白色角化区（有学者认为此型即颗粒型白斑），此型往往是==原位癌==或早期鳞癌。 |

### 四、诊断和鉴别诊断
#### （一）诊断
**典型的天鹅绒样红斑**可作出诊断。==组织病理学检查结果==是**必须的**诊断依据。
对于红斑病损，正规的诊断程序：去除可能的创伤因素如尖锐的牙尖、修复体并==观察 2 周==。如果病损==无明显改善则行病损活检==以明确诊断并排除恶变。
除了活检判断病损的恶性程度外，口腔黏膜自体荧光检查术、口腔黏膜活体染色检查术（如甲苯胺蓝染色）是红斑病临床诊断中可以选用的辅助诊断方法。
病损区域表现为==自体荧光缺失，甲苯胺蓝染色阳性==- 则提示病损的恶变倾向。

#### （二）鉴别诊断

|   疾病    | 内容                                                                                                                                                                                                                                                             |
| :-----: | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

### 五、治疗

1. 口腔红斑病**一旦确诊，立即做根治术**，==手术切除较冷冻治疗临床疗效更为可靠==。
2. 术后复查，建议术后 1 年内每 3 个月复查 1 次。如果复查 1 年后无复发，也未出现其他改变，则复查频率可降为半年 1 次。
3. 一旦出现新发病损，建议立即行活检术以明确性质。

## 第四节 盘状红斑狼疮 DLE

### 一、定义

1. 盘状红斑狼疮 (discoid lupus erythematosus DLE）是一种慢性皮肤 - 黏膜结缔组织疾病。好发于**女性、下唇**。
2. 病损特点：持久性红斑，中央萎缩凹下呈盘状。
3. 累计部位：头面部皮肤及口腔黏膜，皮肤病损表面有黏着性鳞屑，黏膜病损周边有呈**放射状排列的细短白纹**。DLE 亦属于口腔潜在恶性疾患。

### 二、临床表现

临床上 DLE 可分为局限型和播散型。局限型损害仅限于颈部以上的皮肤黏膜，而播散型则可累及颈部以下部位，如上胸、臂、手足背和跖部等。播散型 DLE 患者向 SLE 转变的几率更高。

1. 黏膜损害：
	（1）好发部位：下唇唇红黏膜。唇红和皮肤**界限破坏**。
	（2）典型病损：中央红斑四周有**放射状短条纹**。
	**【鉴别】** 可用于鉴别唇红部的扁平苔藓和糜烂性唇炎病变区亦可超出唇红缘而累及皮肤，唇红与皮肤界限消失，此为 DLE 病损的**特征性表现**。
2. 皮肤损害：
	（1）好发部位：头面部。
	（2）典型病损：常发生在鼻梁和鼻侧以及双侧颧部皮肤所构成的、状似蝴蝶形的区域，故称为“**蝴蝶斑**”
	（3）去除鳞屑可见扩张的毛囊孔，而取下的鳞屑状似“图钉”，即“**角质栓**”。
	（4）DLE 患者对**紫外线**敏感，在受到强烈阳光曝晒后常引发 DLE 急性发作、糜烂加重。
3. 全身症状：
	（1）DLE 患者局部==常无明显自觉症状==，可伴瘙痒、刺痛、灼热等。
	（2）部分患者伴有全身症状，如胃肠道症状、不规则发热、关节酸痛或关节炎、淋巴结肿大、心脏病变、肾脏病变、肝脾大等，对该类患者应进一步检查血常规、尿常规、血沉、心电图、免疫学指标、红斑狼疮细胞等，以排除 SLE 的可能。
4. 儿童 DLE
	（1）DLE 在儿童不常见，其临床特征与成人相似，但**无女性发病较高的趋势**，其光敏感性不明显，发展成 SLE 的可能性较高。

病理：基底细胞液化变性、管周淋巴细胞浸润

### 三、诊断和鉴别诊断

#### （一）诊断

 一般根据皮肤黏膜的病损特点和组织病理学检查诊断。
1. 黏膜病损好发**下唇唇红**，呈圆形或椭圆形红斑或糜烂，中央凹陷，边缘暗红稍隆，病损四周有**白色放射状细纹**。唇部病损常超出唇红边缘而累及皮肤，使黏膜 - 皮肤界限模糊，病损区周围有色素沉着或色素减退。
2. 皮肤病损好发于==头面部==，特征为红斑、鳞屑、毛细血管扩张、毛囊==角质栓==、色素沉着和/或色素减退和瘢痕形成。鼻部周围“**蝴蝶斑**“为其典型表现。
3. **组织活检**具有**重要意义**。免疫荧光检查虽==不是 100% 阳性==，但对诊断及鉴别诊断有意义。
	- 【**直接免疫荧光检查】** 在上皮基底膜区有一连续的、粗细不均匀的翠绿色荧光带，称为 "**狼疮带**"（lupusband）为免疫球蛋白 (IgG、IgM）及补体 C 3 沉积。狼疮带是否存在对该病的诊断、治疗效果及其预后判定具有重要意义。
4. 实验室检查可表现为==血沉加快、y 球蛋白增高==、类风湿因子阳性、抗核杭体阳性、CD 4/CD 8 比率增加等，对诊断具有辅助意义。

### 四、治疗
目前，对于 DLE 虽无根治性疗法，但恰当的治疗可使大多数患者的病情明显缓解，强调早期诊断和早期治疗，以避免转型、毁
容以及癌变的发生。

1. 尽量避免或减少日光照射
2. 局部治疗：
	（1）曲安奈德 + 利多卡因封闭
	（2）下唇血脓肿：0.2% 呋喃西林湿敷
3. 全身治疗：
	（1）羟氯喹：一线药物
	（2）昆明山海棠、雷公藤
	（3）糖皮质激素
