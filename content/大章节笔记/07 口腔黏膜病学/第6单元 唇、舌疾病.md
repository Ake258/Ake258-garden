# 第六单元 唇、舌疾病

慢性唇炎、口角炎、地图舌、沟纹舌、舌乳头炎

## 第一节 慢性非特异性唇炎

### 一、定义

又称**慢性唇炎**（chronic cheilitis)，是不能归入前述各种有特殊病理变化或病因的唇炎，病程迁延，反复发作。
慢性唇炎是唇部慢性、非特异性、炎症性病变。

病因和发病机制：病因不明，可能与温度、化学、机械性因素的==长期持续性刺激==有关；也可能与精神因素有关：患者一般无全身性疾病。
- 干燥、寒冷、舔屑、咬唇、化妆、日照、撕皮、精神因素

### 二、临床表现

分为①脱屑—慢性脱屑性唇炎和②渗出糜烂—慢性糜烂性唇炎。

| 分型                     | 临床表现                                     | 治疗                  |
| ---------------------- | ---------------------------------------- | ------------------- |
| 慢性**脱屑性**唇炎<br>（干燥脱屑型） | 上下唇红部，但以**下唇重**，唇红部干燥、开裂、有黄色或褐色脱屑、脱皮或细鳞屑 | 唇部保湿<br>抗生素软膏或激素类软膏 |
| 慢性**糜烂性**唇炎<br>（湿疹糜烂型) | 上下唇红部**反复糜烂**，渗出明显，结痂剥脱，痂皮脱落形成出血性创面。     | 唇部湿敷                |

![[图6-1 唇炎分类.png|773]]

### 三、诊断及鉴别诊断

诊断：根据病程反复，时轻时重，**寒冷干燥季节**好发，唇红反复干燥、脱屑、痛胀痒、渗出结痂等临床特点，并排除前述各种特异性唇炎后，可作出诊断。

| 与**慢性脱屑性唇炎**<br>鉴别的疾病 | 特征                                                                      |
| :-------------------: | ----------------------------------------------------------------------- |
|         干燥综合征         | 患者也可出现唇红==干燥、皲裂==及不同程度==脱屑==、唇红部呈暗红色等症状。但有**口干、眼干**、合并结缔组织病等其他 SS 典型症状。 |
|        慢性光化性唇炎        | 好发于**日照**强烈的夏季，与暴晒程度有关，脱屑呈秋糠状、**痒感不明显**。                                |
|        念珠菌性唇炎         | 有时表现为唇部干燥脱屑，而不出现假膜红斑糜烂等特征性表现，但常伴有念珠菌口炎和口角炎，实验室检查可发现**白色念珠菌**。           |

| 与**慢性糜烂性唇炎**<br>鉴别的疾病 | 特征                                   |
| :-------------------: | ------------------------------------ |
|      盘状红斑狼疮 DLE       | 好发于唇红缘处，**损害**中央萎缩，外围为白色放射状条纹        |
|       扁平苔藓 OLP        | 唇病损周围白色网状条纹，患者的颊部可见对称性白色网状条纹         |
|         多形性红斑         | 口唇糜烂，常形成褐红色厚痂，自觉疼痛，影响张口及进食。**靶形红斑**。 |

### 四、治疗

原则：避免刺激因素是首要的治疗措施，例如改变咬唇、舔唇等不良习惯，戒除烟酒，忌食辛辣食物，避免风吹、寒冷刺激，保持唇部湿润等。
1. 慢性脱屑性唇炎：
	（1）以**唇部保湿**为主。
	（2）合并感染时可用抗生素软膏或激素类软膏，如金霉素眼膏、氟轻松软膏、曲安奈德乳膏等局部涂布。
2. 慢性糜烂性唇炎：
	（1）应以**唇部湿敷**为主要治疗手段。
	（2）用浸有消毒抗炎液体（如浸有 0.1% 依沙吖啶溶液、3% 硼酸溶液、生理盐水等）的纱布湿敷于患处，==直至结痂消除==，渗出停止，皲裂愈合，==然后涂布软膏==类药物。唇部湿敷联合毫米波、微波、氨氛激光等物理疗法有助于增强局部血液运行、加快药物吸收、提高疗效。

## 第二节 口角炎

### 一、定义

口角炎（angular cheilitis）是发生于**上下唇两侧联合处口角区**的炎症总称，以**皲裂、口角糜烂和结痂**为主要症状。故又称 `口角唇炎`、口角糜烂 (perleche)。

### 二、临床表现

| 项目       | 营养不良性口角炎                | 感染性口角炎                                                                                     | 创伤性口角炎                                | 接触性口角炎                                |
| :------- | :---------------------- | :----------------------------------------------------------------------------------------- | :------------------------------------ | :------------------------------------ |
| **病因**   | 维生素 B 族缺乏               | 由**真菌、细菌、病毒等病原微生物**引起<br>白色念珠菌、链球菌和金黄色葡萄球菌是最常见的感染病原微生物。                                    | 医源性、严重的物理刺激、不良习惯                      | 接触变应原或毒物                              |
| **临床表现** | **双侧口角湿白色，有糜烂**，伴有唇炎、舌炎 | **急性期**：口角区充血、红肿，==有血性或脓性分泌物渗出==，血痂或脓痂，疼痛明显。<br>**慢性期**：增厚呈灰白色，==伴细小横纹或放射状裂纹==，唇红干裂，但疼痛不明显 | 常常为**单侧性**口角区损害，为长短不一的新鲜创口，裂口常有渗血、血痂。 | 口角区充血、水肿、糜烂、皲裂，渗出增多，疼痛剧烈。伴有皮肤表现，全身反应。 |
| **治疗**   | 补充营养及维生素                | 抗病毒、抗菌、抗真菌，去除感染<br>真菌性：首先应提高机体抵抗力。局部以 2%~4% 碳酸氢钠溶液清洗，擦干后涂制霉菌素或克霉唑、咪康唑                       | 局部处理为主，过大的清创缝合                        | 去除过敏源，停止服用可疑药物                        |

## 第三节 地图舌

### 一、定义

地图舌（geographic glossitis）又称 `地图样舌`，是一种浅表性非感染性的舌部炎症，因其表现类似地图上标示的蜿蜒国界而得名。其病损的形态和位置多变，又被称为 `游走性舌炎` (migratory glos-sitis）。
1. 一种浅层**慢性剥脱性舌炎**。
2. 主要舌背、舌缘、舌腹、舌尖。
3. 任何年龄均可发生，==多见于儿童==（6 月~3 岁），一般无自觉症状。

### 二、病因及发病机制

病因尚不明了，多认为与遗传因素有关，常与沟纹舌同时发生。
1. 儿童：==消化不良==、肠寄生虫、维生素 B 缺乏。
2. 成人：==贫血==、胃肠功能紊乱、情绪、病灶。

### 三、临床表现

1. 地图舌好发于==舌背、舌尖、舌缘部==，病损多在==舌前 2/3==，一般不越过==人字沟==。
2. 病损由**周边区**和**中央区**组成。
	（1）中央区表现为丝状乳头**萎缩**呈剥脱样，黏膜表面光滑、充血发红且微凹。故也称**剥脱性舌炎**。
	（2）周边区表现为丝状乳头**增厚**、呈黄白色条带状或弧线状分布，宽约数毫米，与周围正常黏膜形成明晰的分界。
3. 病损==多突然出现==，（小点状逐渐扩大为地图样)，持续 1 周或数周内消退，同时又有新病损出现。因病损的这种萎缩与修复同时发生的特点，使病变位置及形态不断变化，==似在舌背“游走”，可昼夜间发生明显的位置移动==。
4. 地图舌往往有**自限性**，发作一段时间后可有间歇缓解期，此时黏膜恢复如常，间歇期后会再次复发。
5. 地图舌常与**沟纹舌**伴发，地图舌患者口腔的其他部位黏膜也可发生类似地图舌样的病损。
6. 患者一般**无疼痛**等不适感觉，但若合并真菌或细菌感染，食用刺激性食物或饮料时则会有==烧灼样疼痛或钝痛==。

### 四、诊断

根据舌背、舌尖、舌缘等部位的典型地图样改变及其不断变化的游走特征进行诊断，一般不需要进行病理检查。

### 五、治疗

该病**预后良好**。对患者进行病情解释说明和心理疏导，帮助其克服恐惧，是主要的治疗目标。**若无明显不适，一般不需治疗**。若有症状则对症治疗并保持口腔清洁。合并真菌感染者可用 2%～4% 碳酸氢钠溶液含漱。
预防：
	①积极纠正与地图舌有关的发病因素，如调节情绪、避免紧张、劳累和恼怒；
	②积极治疗全身疾病和口腔内病灶；
	③注意饮食卫生、营养均衡；
	④与变态反应有关者应避免食用可能引起变态反应的食物如海鲜、鱼虾、刺激性调味品等。

## 第四节 沟纹舌

### 一、定义
沟纹舌（fissuredtongue）表现为==舌背一条或长或短的中心深沟和多条不规则的副沟==。因沟纹的==形状或排列方向不同==，又称 `脑回舌`
(cerebriform tongue）或 `皱褶舌`(rugae tongue)。发病率低于 10%，无性别差异。
沟纹舌又名 `裂纹舌`，常与**地图舌**同时存在。

### 二、病因

① 病因不明，多认为是先天性发育异常。② 遗传、地理环境、食物种类、B 组维生素缺乏。③ 全身疾病：脓疱性银屑病、梅 - 罗综合征。④ 其他：病毒感染、迟发性超敏反应、苔藓样变

### 三、临床表现

1. 沟纹舌以舌背不同形态、不同排列、不同深浅长短、不同数目的==沟纹或裂纹为特征==，形状似脑回、叶脉或树枝样，也可发生在舌侧缘。
2. 舌尖抵于下前牙舌面将舌拱起，或用前牙轻咬舌体，可清晰见到==张开的沟裂样损害==。
3. 沟底黏膜==连续完整，无渗血==。沟底丝状乳头缺如，沟侧壁丝状乳头稀少。
4. 沟纹舌的部分沟纹可随年龄增长而加重，但进展缓慢，沟纹舌==多伴地图舌==。
5. ==沟纹舌伴有巨唇（肉芽肿唇炎)、面神经麻痹或有面神经麻痹史（MRS）者称==**梅 - 罗综合征**。
6. 患者常无自觉症状，舌的色泽、质地和活动均正常。当沟内食物残渣沉积导致继发真菌或细菌感染时，会出现口臭和疼痛，少数患者可因食物刺激致舌痛、干、苦等。

### 四、诊断

根据舌背沟纹特征可作出诊断

### 五、治疗

无症状、无疼痛者一般**不需治疗**，但应向患者解释该病为**良性**，舌体不会因沟纹的加深而裂穿，以消除患者恐惧心理。
保持口腔清洁，用消炎漱口剂。

## 第五节 舌乳头炎

### 一、定义

舌特殊黏膜表面有==丝状乳头==、==菌状乳头==、==轮廓乳头==和==叶状乳头==；这些乳头发生炎症，称为舌乳头炎 (lingual papillitis)，分别称为==丝状乳头炎==、==菌状乳头炎==、==轮廓乳头炎==和==叶状乳头炎==。
除**丝状乳头炎以萎缩性损害为主**外，其他乳头炎均以充血、红肿、疼痛为主。

### 二、病因及发病机制

1. 局部因素包括残根、残冠、锐利牙尖、牙结石、不良修复体、进食辛辣或过烫食物等刺激及咽部感染。
2. 全身因素包括营养不良、维生素 B 缺乏、贫血、血液性疾病、真菌感染、滥用抗生素、内分泌失调，维生素缺乏等。
3. 叶状乳头在人类口腔中已退化，呈褶皱状位于舌后部的舌缘两侧，==因接触咽部，故咽部的炎症可波及==。

### 三、临床表现

| 临床表现                             | 类型                                                                                                                                                                   |
| :------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 丝状乳头炎                            | ① 丝状乳头遍布于舌背，外观呈白色丝绒状。<br>② 发生炎症时主要表现为丝状乳头发生==萎缩性损害==，严重时可出现==上皮萎缩变薄==，==黏膜充血==。                                                                                      |
| 菌状乳头炎<br>`草莓舌`<br>`杨梅舌`<br>`镜面舌` | ① 菌状乳头分布于舌前部和舌尖部，散在于丝状乳头间，数目较少。<br>② ==炎症时乳头肿胀、充血、灼热、疼痛不适感==。如果菌状乳头肿胀明显，==上皮薄而呈深红==，舌背前部形似 "**草莓 "**，称为 "`草莓舌`" 或 "`杨梅舌`"。<br>" 梅舌 " 有时可见于==猩红热==患者或==黏膜皮肤淋巴结综合征==的舌丝状乳头炎患者。 |
| 轮廓乳头炎                            | ① 轮廓乳头位于舌后 1/3 处，人字沟前方，一般为 7~9 个，呈弧形排列，其侧壁上皮内含味蕾。<br>② 很少发生炎症，炎症时==乳头肿大突起==，==轮廓清晰==，发红。疼痛感不明显，少数患者有味觉迟钝。<br>③ **容易误认为肿瘤**。                                          |
| 叶状乳头炎                            | ① 叶状乳头位于双侧舌缘后部，为 5~8 条纵形并列皱襞，富含淋巴样组织。<br>② 炎症时乳头红肿，乳头间皱褶重显凹陷，患者常有==明显的刺激痛或不适感==，常因担心其会发展为肿瘤而导致不适感加重。<br>③ **容易误认为肿瘤，最好发肿瘤。**                                         |

**镜面舌**：==丝状乳头==和==菌状乳头==同时萎缩，形成==剥脱性红斑==。

### 四、诊断及鉴别诊断

1. 诊断：
	（1）丝状乳头表现为**萎缩**，舌背光滑。（丝 + 菌=镜面舌）
	（2）菌状乳头炎，菌状乳头水肿，充血。（草莓舌、杨梅舌）
	（3）叶状乳头炎表现为红肿，**可发生或误认为**是恶性肿瘤。
	（4）轮廓乳头疼痛感不明显，**易被认为**是肿瘤
2. 鉴别诊断：
	（1）==轮廓乳头炎==易被误认为是肿瘤，==叶状乳头炎==位于舌肿瘤好发区，两者均需与肿瘤鉴别。
	（2）若有潜在恶性疾患史或长期慢性不良刺激史，常伴发渍溃疡和疼痛；触诊局部病损有浸润发硬，且经久不愈，需行组织病理检查，若病理切片有典型的肿瘤表现应诊断为肿瘤。

### 五、治疗

1. ==消除患者思想顾虑==，要求患者戒除频繁伸舌自检的不良习惯。
2. ==去除局部刺激因素==，包括调磨锐利牙尖，清除牙结石，去除不良修复体，减少进食辛辣烫等刺激性食物。
3. 局部可用==抗菌含漱液==，如 0.1% 西吡氯铵、氯已定溶液等。疼痛明显的可用含麻药的含漱液含漱。
4. ==预防舌乳头炎的发生==，应少吃酸、辣、烫等刺激性食物，调磨过锐牙尖，清除牙结石、去除不良修复体，保持口腔清洁，戒除频繁伸舌自检的不良习惯。
5. ==积极治疗咽部的慢性炎症==，清热解毒中成药。
