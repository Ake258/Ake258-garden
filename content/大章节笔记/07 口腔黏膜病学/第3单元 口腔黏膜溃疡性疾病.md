# 第三单元 口腔黏膜溃疡性疾病

## 第一节 复发性阿弗他溃疡

复发性阿弗他溃疡（**RAU**）又称为复发性口腔溃疡（ROU）或复发性阿弗他口炎（RAS）。患病率居口腔黏膜病之首（10%~30%）。
特征：溃疡发作仅限于口腔黏膜，具有**周期性、复发性和自限性**、溃疡灼痛明显的特征。女性高于男性。

### 一、病因

复发性阿弗他溃疡的病因目前尚不清楚。是由多种因素综合作用的结果。
①免疫因素 ②遗传因素 ③系统性疾病因素 ④感染因素 ⑤环境因素 ⑥其他因素

### 二、临床表现

1. 反复发作的溃疡具有 " ==黄、红、凹、痛== " 的临床特征。
2. 长短不一的“==发作期==”：（①==前驱期==+②==溃疡期==）、③“==愈合期==”、④“==间歇期==”
3. 临床上多采用 1968 年 Lehner 分型，将本病分为==轻型、重型及疱疹样阿弗他溃疡==。

|  分类  | 轻型阿弗他溃疡（轻型口疮）                         | 疱疹阿弗他溃疡（疱疹样口疮、口炎型口疮）           | 重型阿弗他溃疡（腺周口疮、复发性坏死性黏膜腺周围炎）                     |
| :--: | :------------------------------------ | :----------------------------- | :--------------------------------------------- |
| 所占比例 | 75%～85%                               | 5%～10%                         | 10%～15%                                        |
| 好发部位 | 唇、舌、颊、软腭等无角化或角化较差的黏膜                  | 唇、舌、颊、软腭等无角化或角化较差的黏膜           | 初始好发于口角，其后于舌腭弓、软硬腭交界处                          |
| 病损特点 | ==直径 5～10 mm==，==3～5 个==散在分布。疼痛**最重** | 小，==约 2 mm==、数目==多==，似 " 满天星 " | 大而深，似 " 弹坑 "，==直径大于 10 mm==，==1～2 个==，疼痛**剧烈** |
|  时间  | 10～14 天愈合                             | 10～14 天愈合                      | 1～2 个月或更长                                      |
| 全身症状 | 无                                     | 头痛、低热等不适、局部的淋巴结肿痛              | 低热乏力、局部区域的淋巴结肿痛                                |
|  愈后  | **无**瘢痕                               | **无**瘢痕                        | **可留瘢痕**                                       |

### 三、诊断及鉴别诊断

（1）诊断　根据病史（周期性、复发性和自限性）和临床体征（黄、红、凹、痛）。
（2）鉴别诊断（重点记忆）
	① ==疱疹样阿弗他溃疡与急性疱疹性龈口炎==鉴别。
	② 白塞病（Behcet's disease）又称**口、眼、生殖器三联征**。除口腔溃疡外，有生殖器黏膜溃疡和眼部病变。  
		皮肤损害：结节性红斑，毛囊炎及针刺反应阳性。  
	③ 重型口疮需与==创伤性溃疡、恶性肿瘤溃疡、结核性溃疡==相鉴别。

| 项目        | 疱疹样阿弗他溃疡          | 急性疱疹性龈口炎（原发性疱疹性口炎）          |
| :-------- | :---------------- | :-------------------------- |
| 好发年龄      | 中青年               | 婴幼儿                         |
| 病损特点      | 十余个至几十个散在小溃疡，无发疱期 | 成簇小水疱，破后成表浅溃疡，可融合           |
| 病损部位      | 非角化黏膜多见           | 牙龈、硬腭等角化黏膜                  |
| 发病特点及全身反应 | **反复发作**，全身反应较轻   | **急性发作**，全身反应较重、可出现低热、淋巴结肿大 |
| 皮肤损害      | 仅局限口腔黏膜、无皮肤损害     | 可伴皮肤损害                      |

| 项目       | 内容                                                                                     |
| :------- | :------------------------------------------------------------------------------------- |
| **重型口疮** | 溃疡反复发作，形状规则，有自限性                                                                       |
| 创伤性溃疡    | 形状不规则，常==与损伤因素基本契合==，周围有炎症性增生反应，黏膜发白                                                   |
| 恶性肿瘤溃疡   | ==深大==，表现为==边缘隆起的火山口状==溃疡病变进展迅速，基底有硬结扪诊为==不可移动硬块==，无自限性。                               |
| 结核性溃疡    | 为口腔中最常见的继发性结核损害，结核分枝杆菌，溃疡边缘微隆，呈==鼠啮状，潜掘状==边缘，基底有桑葚状肉芽组织增生，==无自限性==。<br>确诊：组织病检，朗格汉斯巨细胞 |

### 四、治疗

主要目标：减少复发次数、延长间歇期、减轻疼痛、促进愈合。
治疗原则：①全身治疗和局部治疗相结合 ②中医治疗和西医治疗相结合 ③生理治疗和心理治疗相结合；以消除病因、增强体质、对症治疗。
（1）药物治疗：
	1. 局部用药：
		①止痛药物一**达克罗宁、利多卡因**
		②糖皮质激素一曲安奈德、地塞米松
		③腐蚀性药物—10% 硝酸银、50% 三氛醋酸、95% 乙醇、8% 氯己定
		④局部封闭一曲安奈德 + 利多卡因
	2. 全身用药
		①肾上腺皮质激素
		②免疫抑制剂——细胞毒素类药物：环磷酰胺、沙立度安
		③免疫增强剂——口服左旋咪唑、注射转移因素、卡介苗
		④维生素 C 复合
		免疫球蛋白 - 被动
（2）超声雾化治疗
（3）物理治疗
（4）中医中药

## 第二节 白塞病（Behcet's disease,BD）

白塞病（**BD**）又称白塞综合征、贝赫切特综合症，中医成为“狐惑病”。是一种慢性血管炎症性疾病。因同时或先后发生的口腔黏膜溃疡以及眼、生殖器、皮肤病损是该病的主要临床特征，几乎累及每一病理，而被称为“**口 - 眼 - 生殖器三联症**”。临床表现也可累及血管、神经系统，消化道、关节、肺、肾、附睾等器官，为一系统性疾病。
==是以细小血管炎为病理基础的慢性进行性、复发性、系统性损害性疾病，与自身免疫病有关。==

### 一、病因

本病确切病因尚不明确，可能与遗传、感染、免疫异常等因素有关。

### 二、临床表现

本病全身各系统均可受累，但多种临床表现较少同时出现，有时须经历数年甚至更长的时间才相继出现。

#### （一）常见症状

（1）**复发性口腔溃疡**（白塞病的首发且必发症状）
（2）皮肤损害：①结节性红斑 ②毛囊炎 ③针刺反应阳性（不出血，出脓）
（3）生殖器溃疡
（4）眼炎：葡萄膜炎和视网膜血管炎

#### （二）少见症状

关节、血管、神经、消化道

|     部位     | 特征                 |
| :--------: | ------------------ |
|     关节     | 与风湿性关节炎的症状相似，有红肿热痛 |
|    血管损害    |                    |
|   消化道损害    | 又称肠白塞病             |
|    神经损害    | 精神错乱综合征            |
|    肺部损害    | 咳嗽、咯血、胸痛、呼吸困难等     |
| 肾脏、心脏等其他损害 |                    |

### 三、诊断及鉴别诊断

诊断：**复发性口腔溃疡**+==针刺反应阳性==/皮肤损害/生殖器溃疡/眼炎 （1+2 即可诊断）

### 四、治疗

|  治疗  | 病患       | 方案                                                 |
| :--: | -------- | -------------------------------------------------- |

## 第三节 创伤性血疱和创伤性溃疡

创伤性血疱（traumatic mucosal heatoma）和创伤性溃疡（traumatic ulceration) 是由物理性、机械性或化学性刺激引起的病因明确的口腔黏膜损害。当刺激因素较强，机体反应较迅速时可引起**血疱**，长期慢性刺激则可引起**溃疡**。黏膜血疱一旦破溃和继发感染，则发生糜烂或者溃疡。

### 一、病因

#### （一）创伤性血疱
因食用过烫食物、咀嚼大块干硬食物或吞咽过快而擦伤口腔黏膜，会引起创伤性血疱，可因外力挫伤或误咬颊、舌黏膜造成血疱。又称黏膜血疱 (mucosal hematoma）。

#### （二）创伤性溃疡
（1）非自伤：残根残冠、尖锐边缘嵴，不良修复体。
（2）自伤：下意识咬唇、咬颊、铅笔尖刺、婴幼儿吸吮拇指、橡胶乳头、玩具等。
（3）化学性灼伤：误服强酸强碱，或三氧化二砷、硝酸银灼伤黏膜。
（4）热冷刺激：过烫或过冰的食物。
特点：**慢性、深大的溃疡**，周围有炎症增生反应，黏膜水肿明显。

#### 二、临床表现

|            项目             | 病因                                    | 临床表现                                                                                                                                                            |
| :-----------------------: | :------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|       **压（褥）疮性溃疡**        | 由持久的非自伤性机械性刺激造成。多见于老年人。               | 义齿边缘处有溃疡**且**可见有==组织增生==。<br>残根残冠或不良修复体长期损伤黏膜<br>溃疡深及黏膜下层，边缘轻度隆起，色泽灰白，疼痛不明显。                                                                                    |
|  **Bednar 溃疡<br>（贝氏口疮）**  | 因婴儿吸吮拇指、用过硬的橡皮奶头引起                    | 固定发生于**硬腭、双侧上腭翼钩处**黏膜表面，双侧对称分布，摩擦，易溃疡。溃疡表浅，婴儿哭闹不安、拒食。                                                                                                           |
| **Riga-Fede 溃疡<br>李 - 弗溃疡** | 专指发生于**婴幼儿舌腹**的溃疡。                    | 因过短的舌系带和过锐的**新萌乳下中切牙**长期摩擦引起，舌系带、舌腹处充血、肿胀、溃疡。久不治则转变为肉芽肿性溃疡，们诊有坚韧感，影响舌活动。                                                                                        |
|           自伤性溃疡           | 好发于性情好动的青少年或患多动症的儿童，患者常有用铅笔尖捅刺黏膜不良习惯。 | 1. ==右利手者==，溃疡好发于左侧颊脂垫尖或磨牙后垫处，==左利手者==，溃疡位置反之<br>2. 有咬唇、咬颊、咬舌不良习惯者，溃疡好发于下唇黏膜或两颊、舌背，溃疡深在，长期不愈<br>3. 溃疡外形不规则，周围因为长期的机械性刺激导致白色斑块。<br>4. ==基底略硬或有肉芽组织==，疼痛不明显，有时有痒感 |

### 三、诊断及鉴别诊断

#### （一）诊断

有明确的理化刺激因素或自伤、灼伤等病史，创伤性溃疡的部位和形态往往与机，械性刺激因子相符合。无复发史。去除刺激因素后，溃疡很快明显好转或愈合。若长期不愈者应作活体组织检查。

#### （二）鉴别诊断
对去除刺激因素后仍长期不愈的深溃疡应与一些特异性深溃疡鉴别。

|   疾病   | 临床表现                                                                                    |
| :----: | :-------------------------------------------------------------------------------------- |
|  腺周口疮  | 溃疡深大（弹坑样），常伴发小溃疡，有反复发作史，无创伤史和自伤性不良习惯，口内无机械性刺激因素存在。==愈合后留有瘢痕。==                          |
| 结核性溃疡  | 溃疡深凹，边缘呈==鼠噬状==，基底高低不平，呈==粟粒状==小结节，有==红色肉芽组织==。伴低热、==盗汗==、==淋巴结肿大==。结核菌素试验阳性。==无创伤因素==。 |
|  癌性溃疡  | 常为鳞状细胞癌。溃疡深大，底部有==菜花状==细小颗粒突起，==边缘隆起翻卷==，扪诊有==基底硬结==，疼痛不明显。                             |
| 化脓性肉芽肿 | 好发于==牙龈==，表现为深红色半球形肿块，有时表面形成溃疡。组织病理特征为血管增生性肉芽肿。                                         |

### 四、治疗
1. **尽快去除刺激因素**是**首要措施**
	- 包括拔除残根，残冠，磨改过锐牙尖和边缘崤，修改不良修复体，纠正咬唇、咬颊、咬舌不良习惯，改变婴儿喂食方式（不用奶瓶改用小匙喂食）手术矫正舌系带过短等。
	- 其次是局部涂敷复方皮质散、养阴生肌散、冰硼散等消炎防腐药物; 含漱氯已定液、复方硼酸液等，以防继发感染。
	- 对有全身症状或继发感染者应服用抗生素。==长期不愈的深大溃疡应作==**活检**，==排除癌变。
2. 预防避免不良理化因素的刺激，养成良好进食习惯。定期检查口腔牙颌状况，避免口腔治疗中的操作失误。正确使用药物。
