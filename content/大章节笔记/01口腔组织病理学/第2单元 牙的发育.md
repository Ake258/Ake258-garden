# 第二单元 牙的发育

## 第一节 牙胚的发生和分化

### 一、牙胚

（1）牙胚是由**牙板（外胚层）**+ 邻近的**外胚间叶组织**发育而来。  
（2）牙胚的分化：牙胚是牙发育的始基，由 [[成釉器]]、[[牙乳头]] 及 [[牙囊]] 三部分组成。  
- **成釉器**来源于==口腔外胚层==（牙胚当中最先形成），**牙乳头及牙囊**来源于==外胚间叶==。

|   发育时间    | 牙胚结构 |   来源    |    发育牙体结构     |
| :-------: | :--: | :-----: | :-----------: |
|  胚胎第 8 周  | 成釉器  | ==外胚层== |      牙釉质      |
| 胚胎 9~10 周 | 牙乳头  |  外胚间充质  |    牙髓、牙本质     |
|           |  牙囊  |  外胚间充质  | 牙周膜、固有牙槽骨、牙骨质 |

牙胚在胚胎第 8 周开始发育，胚胎 9~10 w 牙胚**真正形成**（三部分凑齐），11~12 w 钟状期成釉器。

### 二、牙板

![[图2-1 牙胚的发生.png]]
- **勘误**
胚胎第 5 周
胚胎第 7 周
胚胎第 8 周→牙板上皮→成釉器（花蕾）
胚胎第 9~10 周→成釉器 + 牙乳头 + 牙囊=完整的牙胚的构成；由于牙乳头形成，成釉器形成帽状结构，帽状期成釉器。
胚胎第 11~12 周→牙乳头逐渐成熟，成釉器形成倒扣钟样凹陷，钟状期成釉器。

### 三、成釉器

牙胚在胚胎第 8 周开始发育，胚胎 9~10 w 牙胚**真正形成**，11~12 w 钟状期成釉器。
成釉器的发育是一个连续的过程，可分为三个时期：**蕾状期**、**帽状期**和**钟状期**。

|      成釉器时期       |            时间            | 层次                                                                                                                                                                                               |
| :--------------: | :----------------------: | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|     **蕾状期**      |       ==胚胎第 8 周==        | **无**细胞分层，**无**功能，牙板的 20 个定点                                                                                                                                                                     |
| **帽状期**<br>（增殖期） | ==胚胎第 9～10 周==<br>可见支持组织 | 分化为**三层**：<br>①外釉上皮：呈==立方形==<br>②星网状层：排列疏松，呈==星形细胞==，起**营养和缓冲**内釉上皮作用。<br>③内釉上皮：==矮柱状==，晚期高柱状。<br>**颈环**（cervical loop）: 内釉上皮与外釉上皮相连处称颈环，在上皮根鞘的发生中起重要作用。                                          |
|     **钟状期**      |     ==胚胎第 11～12 周==      | 分化为四层：<br>①外釉上皮（单层排列立方状细胞，==晚期呈==**皱褶样**==排列==）；<br>②星网状层；（桥粒连接成网状，**营养缓冲作用**）<br>③中间层（==2~3 层扁平细胞==，具有高的碱性磷酸酶活性，激活内釉上皮层**与釉质形成有关**）。<br>④内釉上皮（单层，早期==低柱==状→晚期==高柱==状，分化成釉细胞）；<br>**钟状晚期开始形成釉质** |

牙胚在**帽状期**形成：  
①乳牙牙胚形成于==胚胎第 9～10 周==；  
②恒牙牙胚形成于==胚胎的第 4 个月==，钙化于出生时
![[图2-2 成釉器的分期.png]]
在相邻的==内釉细胞==之间，有一特化的附着结构，即一种**细胞连接复合体**（junctional complex）进入细胞的胞浆中，形成**终棒**（termi-nal bar），在釉质形成中起重要作用，为釉质的形成做好了准备。

- 牙釉质表面的==特殊==结构
	1. **釉珠**：为局限性釉质增生过长的小球形团块。常发生在==磨牙根分叉处==的牙骨质表面，接近釉牙骨质界（牙颈部）。  
		- 须与==釉结==鉴别
- ==帽状期==和==钟状期==牙胚可出现一些==暂时性==的结构，牙真正形成时消失。
	1. **釉结**（enamel knot）：牙胚内釉上皮中央，局部聚集（簇状）的未分化上皮细胞。
		- **牙发育的组织中心，每个牙胚只有一个**==原发性釉结==：**调控牙尖形态**（决定牙尖形态的是牙乳头）。
		- 当原发釉结消失后，在磨牙未来的牙尖顶部将出现==继发釉结==。
	2. 釉龛（enamel niche）：片状压板向内凹陷形成腔隙，切片表现为两个上皮条索与口腔上皮相连。(了解即可)

### 四、牙板结局

**Serre 上皮剩余**（Serre epithelial rest）：**钟状期末**，牙板部分上皮以上皮岛和上皮团的形式残留在==颌骨或黏膜==中，称为 Serre 上皮剩余。
**马牙**/**上皮珠**（epithe-lial pearls）：本质 -==Serre 上皮剩余==。婴儿出生后不久，可在==牙龈上出现针头大小的白色突起==，即为角化的==上皮珠==，可自行脱落。
有时残留的上皮可成为牙源性肿瘤或囊肿的上皮来源。

## 第二节 牙体及牙周组织的形成
牙硬组织的形成从**生长中心**开始

| 牙齿  |      生长中心       |
| :-: | :-------------: |
| 前牙  | **切缘**和舌侧隆突的基底膜 |
| 磨牙  |     **牙尖**      |

牙体及牙周组织的形成过程：**最先**开始形成→**牙本质**→==牙釉质==→==牙骨质==、==牙周膜==及==牙槽骨内壁==。

![[图2-4 牙本质沉积.png]]

### 一、牙本质的形成

1. 内釉上皮分化形成**成釉细胞**，成釉细胞**诱导**牙乳头分化出**成牙本质细胞**。  
	- ==成釉细胞==→未分化间充质细胞→==前成牙本质细胞==→==成牙本质细胞==→**牙本质**
	- 牙本质刺激成釉细胞成熟→形成**牙釉质**
2. ==成牙本质细胞==：切缘、牙尖（柱状），牙颈部（立方状）。
3. **最早**、最外的牙本质基质为**罩牙本质** ；向内继续沉积为**髓周牙本质**；最内层（靠近牙髓）为**前期牙本质**
4. 牙本质的==矿化形式==是以==球形矿化==为主。
	1. 球形钙化（主要、生长期）
	2. 线性钙化（静止期）

![[图2-5 牙本质的形成.png]]

### 二、牙釉质的形成

（1）牙釉质的形成包括 2 个阶段。

|  阶段  | 时期  | 过程                       | 无机物占比   |
| :--: | :-- | :----------------------- | :------ |
| 第一阶段 | 分泌期 | 1.==产生基质==；<br>2.==初矿化== | **30%** |
| 第二阶段 | 成熟期 | ==釉质进一步矿化（再矿化）==         | **96%** |

（2）正常情况下每天分泌釉基质的厚度约为 **4μm**，表层矿化 - 快。每两天之间基质的界限即为**釉柱横纹**。  
（3）成釉细胞由**内釉上皮**形成。牙釉质和牙本质形成过程有严格的==规律性==与==节拍性==，==交叉进行==，牙本质向牙髓中央后退，牙釉质向外周后退。
（4）**托姆斯突**：又称==成釉细胞突==，无釉柱釉质形成后，成釉细胞开始离开牙本质表面，在靠近**釉牙本质界**的一端，形成一短的圆锥状突起。  
（5）终棒和连接复合体：位于托姆斯突和细胞体之间。  
（6）极性倒置：细胞核**远离基底膜**。
（7）**釉小皮**：牙冠形成后，成釉细胞在釉质表面分泌一层无结构的有机薄膜，与成釉器分隔开。
（8）釉质发育完成后==成釉细胞、中间层细胞和星网状层与外釉上皮细胞==结合，形成的一层鳞状上皮覆盖在==釉小皮==（无结构）上，称为**缩余釉上皮**（4 层变 1 层），牙颈部将来演化为牙龈的**结合上皮**，桥粒连接在釉小皮上。

### 三、牙根的形成及牙周组织的发育

| 牙根的形成 | 决定结构       |
| :---: | :--------- |
| 牙尖的数量 | **牙乳头**    |
| 牙尖的形态 | **釉结**     |
| 牙根的数量 | **上皮隔**    |
| 牙根的形态 | **上皮根鞘**   |
| 侧支根管  | **上皮根鞘断裂** |

① 釉质发育完成后==成釉细胞、中间层细胞和星网状层与外釉上皮细胞==结合，形成的一层鳞状上皮覆盖在釉小皮（无结构）上，称为**缩余釉上皮**（4 层变 1 层），牙颈部将来演化为牙龈的**结合上皮**，桥粒连接在釉小皮上。  
② **上皮根鞘**：牙冠即将完成发育时，牙根开始发生，内釉和外釉上皮细胞在**颈环**处向未来根尖孔方向增生形成**上皮根鞘**，这样增生的上皮呈双层，上皮根鞘和邻近的外胚间叶细胞决定着将来牙根的数量、长度和形态等。
③**上皮隔**：上皮根鞘继续生长，离开牙冠向牙髓方向呈约 45° 角弯曲，形成一盘状结构。弯曲的这一部分上皮称上皮隔，决定牙根的数量。
④上皮根鞘诱导根部==牙乳头==形成**根部牙本质**，过程中上皮根鞘==异常断裂==会缺失牙本质，形成了**侧支根管**。
⑤牙囊细胞在==上皮根鞘逐渐断裂消失后==进入上皮根鞘内侧形成**牙骨质**，若一部分上皮根鞘异常没有完全断裂会影响牙骨质形成，造成**牙骨质缺如**，**牙本质过敏**。
⑥上皮根鞘最后部分消失，剩余的上皮根鞘残留保留在==牙周膜中==，形成**马拉瑟（Malassez）上皮剩余**。

![[图2-6 牙釉质的形成、牙根的形成.png]]

### 四、牙髓的发生
原发性牙本质完全形成→牙乳头的未分化间充质细胞→纤维细胞（星形）压碎细胞→余留在髓腔内的多血管的结缔组织→**牙髓**

==牙髓牙本质复合体==：牙本质和牙髓由于胚胎发生和功能上相互关系密切，二者的合称。

### 五、牙周膜与牙槽骨的形成
牙周膜的形成：牙囊→成纤维细胞→一端附着于牙骨质，一端附着于牙槽骨→牙周膜
牙槽骨的形成：牙囊→成骨细胞→骨

## 第三节 牙的萌出和替换

**牙的萌出**（eruption）是指发育中的牙在牙冠形成后向𬌗平面移动，穿过骨隐窝和口腔被膜，出现在口腔中，并达到咬合平面的一个复杂的过程。
==三个时期==：萌出前期、萌出期、萌出后期
**被动萌出**，伴随着牙齿的主动萌出，缩余釉上皮与釉小皮表面分离，临床==牙冠暴露，牙龈向根方移动==。

|          结构           | 来源    | 位置    |
| :-------------------: | :---- | :---- |
|    **Serre 上皮剩余**     | 牙板    | 颌骨或牙龈 |
| **马拉瑟（Malassez）上皮剩余** | 上皮根鞘  | 牙周膜   |
|       **缩余釉上皮**       | 缩余釉上皮 | 结合上皮  |
