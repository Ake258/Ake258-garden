# 第五单元 口腔黏膜

## 第一节 口腔黏膜的基本结构

### 一、上皮

#### （一）角化上皮

角化的复层鳞状上皮主要由角质细胞构成，由深部至表面可分为四层（**基底层、棘层、颗粒层、角化层**）。

| 分层  | 特点                                                                        |
| --- | ------------------------------------------------------------------------- |
| 基底层 | 最深部，借基底膜与固有层结缔组织相连。基底细胞与邻近的棘层==具有分裂增殖能力==，因此被称为**生发层**                    |
| 棘层  | 由体积较大的多边形细胞构成，在上皮中是**层次最多**的细胞层，常伸出多而小的棘状突起与相邻细胞连接，称为**细胞间桥**。作用是维持上皮的完整性 |
| 颗粒层 | 2～3 层扁平细胞组成，胞质中有**嗜碱性透明角质颗粒**                                             |
| 角化层 | 位于表层，角化细胞中胞核完全消失者称为**正角化**，如果含有浓缩而未消失的细胞核者，称为**不全角化**                     |

#### （二）非角化上皮

由深部至表面可分为**基底层、棘层、中间层、表层**。

| 分层  | 特点                               |
| --- | -------------------------------- |
| 基底层 | 细胞与角化上皮相似                        |
| 棘层  | 细胞体积较大，细胞**间桥**==不明显==           |
| 中间层 | 细胞排列更紧，细胞**间隙**==不明显==，为棘层和表层的过渡 |
| 表层  | **有细胞核**的扁平细胞                    |

#### （三）非角质形成细胞

包括**黑色素细胞**、**朗格汉斯细胞**和**梅克尔细胞**。细胞内没有张力细丝和桥粒，在普通切片中细胞质不着色，所以又称为**透明细胞**。

| 名称     | 形态   | 分布    | 功能         | 来源       |
| ------ | ---- | ----- | ---------- | -------- |
| 黑色素细胞  | 树枝状  | 基底层   | 产黑色素       | 神经嵴细胞    |
| 郎格汉斯细胞 | 树枝状  | 主要在棘层 | ==与免疫有关==  | 造血组织     |
| 梅克尔细胞  | 无树枝状 | 基底层   | ==压力感受细胞== | 神经嵴或上皮细胞 |

### 二、口腔黏膜的基本结构

| 基本结构 | 特点                                                                     |
| ---- | ---------------------------------------------------------------------- |
| 上皮   | 复层鳞状上皮，分为角化和非角化复层鳞状上皮                                                  |
| 基底膜  | 电镜下基底膜由**透明板**、**密板**和**网板**构成。上皮和基底膜以**半桥粒**的方式结合在一起                  |
| 固有层  | 纤维主要是==Ⅰ型胶原纤维==                                                        |
| 黏膜下层 | 由疏松结缔组织所构成，内含==小唾液腺、较大的血管、淋巴管、神经以及脂肪组织==。<br>在牙龈、硬腭的大部分区域和舌背，==无黏膜下层== |

## 第二节 口腔黏膜的分类

口腔黏膜根据部位和功能可分为三种类型：**咀嚼黏膜**（牙龈、硬腭）、**被覆黏膜**和**特殊黏膜**（舌背黏膜）。

| 分类       | 分布           | 特点                                |
| -------- | ------------ | --------------------------------- |
| **咀嚼黏膜** | 硬腭、牙龈        | ①有角化（角化层、颗粒层、棘层、基底层）；<br>②大多无黏膜下层 |
| **被覆黏膜** | 唇、颊、口底、舌腹、软腭 | ①无角化（表层、中间层、棘层、基底层）；<br>②黏膜下层厚    |
| **特殊黏膜** | 舌背           | ①有四种乳头、有味蕾；<br>②无黏膜下层             |

（1）唇：唇红有角化（是唯一有角化的被覆黏膜），黏膜下层没有唾液腺和皮脂腺，故易干裂。  
（2）颊：在口角后区可见成簇的粟粒状淡黄色小颗粒，即异位皮脂腺，称为福代斯斑。  
（3）舌乳头：按其形态可分为以下几种。

| 名称   | 分布与特点                                | 生理作用     | 病理状态                 |
| ---- | ------------------------------------ | -------- | -------------------- |
| 丝状乳头 | 遍布舌背，舌尖最多；数量**最多**，个体**最小**，分布**最广** | 无味蕾，构成舌苔 | ==地图舌、镜面舌==          |
| 菌状乳头 | 舌尖和舌侧缘，分散于丝状乳头之间                     | ==甜==味   | ==草莓舌==              |
| 轮廓乳头 | 界沟前方，约 10 个，**体积最大，数目最少**            | ==苦==味   | 炎症                   |
| 叶状乳头 | 舌侧缘后部。**退化**为 5～8 条平行排列的皱襞           | ==酸==味   | 当发炎肿痛时，==常被疑为肿瘤==而就医 |

## 第三节 口腔黏膜临床病损特点（病损的基本类型）

### 一、斑和斑片

斑（macule）＜==2 cm==＜斑片（patch）

#### 特点

1. 性质：颜色异常的局限损害（==不高==、不厚、不硬）
2. 颜色：较深（红色、红棕色、棕黑色）
3. 外形：圆形、椭圆形、线性
4. 质地、外形无改变

### 二、丘疹和丘斑（斑块）

#### （一）丘疹（papule）

1. 性质：小的实体性==突起==，针头大，直径**小于 1 cm**
2. 形状：基底——圆、椭圆形；表面——尖、圆、扁平形；颜色：灰白色、红色，消退后不留痕迹
3. 口腔粘膜的丘疹：由大量排列不一病损组成
4. 口腔扁平苔癣：典型的丘疹排列成带状、斑块、环状

#### （二）丘斑（plaque）

1. 性质：界限清楚，稍隆起、坚实的病损，直径大于 **1cm**
2. 大小：不等
3. 颜色：白色或灰白色
4. 表面：比较平滑或粗糙、有沟裂分割
5. 口腔粘膜白斑、癌、慢性盘状红斑狼疮

### 三、疱和大疱

疱：黏膜内贮存液体而成（圆形、突起、半透明）
疱（vesicle）＜**1 cm**＜大疱（bulla）
可根据内容物不同分为①脓疱 ②血疱 ③水疱 。例如脓疱（pustule）内容物由脓性物取代了透明的疱液。

#### （一）病理相关

1. 上皮**内**疱（棘层内疱）：疱壁薄软
2. 上皮**下**疱（基层下疱）：疱壁厚
3. 发展：疱壁破裂——糜烂 或 溃疡
4. 病因：疱性损害可见于：病毒感染、药物反应、烫伤、疱性皮肤病

### 四、糜烂和溃疡

#### （一）糜烂（erosion）

1. 性质：粘膜的表浅缺损，上皮的==部分损伤==，不损及基底细胞层
2. 颜色：红色
3. 大小：形状不定、边界不清、表面光滑

#### （二）溃疡（ulcer）

1. 性质：粘膜上皮==完整性==发生持续性缺损或破坏
2. 形状：凹陷（表层坏死脱落）
3. 浅层溃疡：只破坏上皮层，愈后无瘢痕
4. 深层溃疡：波及粘膜下层，愈后遗留瘢痕
5. 溃疡底部：结缔组织和纤维蛋白

### 五、结节和肿瘤

#### （一）结节（nodule）

1. 性质：突起于口腔黏膜的实体病损（结缔组织成分）
2. 大小：一般直径为 5cm，形状不定，大小不等。
3. 颜色：粉红至深紫色
4. 见于：纤维瘤、痣

#### （二）肿瘤（tumor）

起自黏膜向外突起的实体性生长物，其大小、形状、颜色不等。
真性肿瘤：良性、恶性
	良性肿瘤表面较规则，触诊时比较活动。
	恶性肿瘤常较固定，表面常不规则并有溃疡。
瘤样病变：血管瘤、肉芽肿、囊肿

### 六、萎缩（atrophy）

1. 性质：表面所覆盖的上皮变薄（组织细胞体积变小，数量不减少）
2. 颜色：==发红==的病变
3. 形状：略呈**凹陷**，特有的一些上皮结构消失，被一薄层上皮取代。
4. 见于：舌乳头萎缩，可使舌面光滑而发红。

### 七、皲裂（rhagades）

1. 性质：粘膜表面的线状裂口，由炎性浸润使组织**失去弹性变脆**。
2. 见于：**核黄素**缺乏——口角皲裂
3. 深度：
	上皮内：痊愈后不留瘢痕
	粘膜下层：愈合后有瘢痕，出血、灼痛

### 八、假膜（pseudomembrane）

1. 性质：由炎性渗出的纤维素、坏死脱落的上皮细胞、炎性细胞聚集而成
2. 颜色：灰白色或黄白色膜；见于溃疡表面、可结痂
3. 它不是组织本身，故**可以擦掉或撕脱**

### 九、坏死和坏疽

#### （一）坏死（necrosis）

体内局部细胞病理性死亡：如坏死性龈口炎、复发坏死性黏膜腺周围炎、白血病的牙龈

#### （二）坏疽（gangrene）

较大范围的坏死，又受腐物寄生菌作用而发生腐败
	大范围坏死＋腐物寄生菌，如==走马牙疳==（坏死性口炎）
