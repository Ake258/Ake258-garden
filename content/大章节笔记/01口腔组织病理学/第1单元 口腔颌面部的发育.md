# 第一单元 口腔颌面部的发育

## 第一节 鳃弓和神经嵴

### 一、鳃弓（咽弓）
**鳃弓**：第四周时，原始咽部的间叶细胞迅速增生，形成左右对称的背腹走向的 ==6 对柱状隆起==，与 6 对主动脉弓相对应。

| 鳃弓  | 名称        |
| --- | --------- |
| 第一对 | **下颌弓**最大 |
| 第二对 | **舌弓**    |
| 第三对 | **舌咽弓**   |

> 鳃弓有 6 对，其余 3 对无特别的名称。

![[图1-1 咽囊的结构、鳃弓与面突.png]]

### 二、发育异常
**颈窦**：==第 2 腮弓==生长速度快，朝向胚胎的尾端，覆盖了 2、3、4 腮沟和 3、4、5 腮弓并与颈部组织融合；被覆盖的腮沟与外界隔离，形成一个由==外胚层==覆盖的腔称颈窦。
**耳前窦道**：==第 1 腮沟==和第 1、2 腮弓发育异常时耳屏前方形成皮肤的狭窄盲管或点状凹陷。（一端通）
**耳前瘘管**：耳前窦道继续向深部延长，与==鼓室==相通。（两端通）

|      异常       | 主要相关异常结构            |
| :-----------: | :------------------ |
|    ==颈窦==     | ==第 2 鳃弓==          |
| ==耳前窦道/耳前瘘管== | ==第 1 鳃沟和第 1、2 鳃弓== |

![[图1-2 颈窦、耳前瘘管.png|558]]

### 三、神经嵴（neural crest）

口腔的上皮组织、牙本质、牙髓、牙骨质等都是来源于神经嵴细胞（==外胚间充质==），不包括**牙釉质**（外胚层）。

## 第二节 面部的发育

面部的发育来自于**第 1 鳃弓**和**额鼻突**。

### 一、面部发育过程

面部是由下颌突、上颌突、侧鼻突和中鼻突（包括球状突）联合而形成的。

- 第一时期：增殖期（受孕至受孕后 2 周）。
- 第二时期：==胚胎期（3～8 周）→口腔颌面部发育基本在此期完成==。
- 第三时期：胎儿期（9 周开始）→腭部发育在此期开始阶段完成。

| 时间                | 面突发育过程              | 特点            |
| ----------------- | ------------------- | ------------- |
| ==胚胎第 3 周==       | **额鼻突**、==第一鳃弓==、口凹 |               |
| 胚胎第 3 周末          | **拉特克囊**            | 口咽膜开始破裂       |
| 胚胎第 4 周初（24 天）    | 下颌突➡上颌突             | 口咽膜==完全破裂==   |
| 胚胎第 4 周末          | 额鼻突➡中鼻突 x 1、侧鼻突 x 2 | 口咽膜==完全破裂==   |
| 胚胎第 5 周           | 中鼻突➡球状突（形成人中）       | 面部发育所需突起已齐备   |
| ==胚胎第 6 周和第 7 周== | 突起开始联合、融合           | 各种致畸因子可形成面部畸形 |
| ==胚胎第 8 周==       | **初具人的面形**          |               |

**拉特克囊（Rathke pouch):**
约在胚胎==第 3 周末==，在口咽膜前方口凹顶端正中出现个囊样内陷，此囊不断加深，囊中的==外胚层细胞==增生并向前脑腹侧面移动并分化出==垂体前叶细胞==，此后囊退化消失，此囊的残余可发生颅咽管瘤。
**面突的联合（merge)**：面部的进一步发育， 突起之间的沟就会随着面突的生长而变浅、消失。（两突起原是一个结构，突起间的沟变浅而融合）
**面突的融合（fuse)**: 突起和突起之间在生长过程中发生表面的外胚层互接触、破裂、退化、消失。（两个独立突起，突起相互接触后融合）

![[图1-3 面部发育示意图.png|769]]

| 第 24 天 | 上界额鼻突、两侧上颌突、==下界第一鳃弓（下颌突）== |

### 三、面突及其衍生组织

| 起源       | 突起           | 软组织形成物                        | 硬组织形成物                |
| -------- | ------------ | ----------------------------- | --------------------- |
| **额鼻突**  | ==中鼻突（球状突）== | 鼻梁、鼻尖、鼻中隔各软组织、上颌切牙牙龈、腭乳头、上唇中部 | 筛骨、犁骨、前颌骨、==上颌切牙==、鼻骨 |
|          | ==侧鼻突==      | 鼻侧面、鼻翼、部分面颊                   | 上颌骨额突、泪骨              |
| **第一鳃弓** | ==上颌突==      | 上唇、上颌后牙牙龈、部分面颊                | 上颌骨、颧骨、腭骨、==上颌磨牙及尖牙== |
|          | ==下颌突==      | 下唇、下颌牙龈、面颊下部                  | 下颌骨、==下颌牙==           |

![[图1-5 面部的发育.png|494]]

### 四、面部发育异常

在胚胎第 6～第 7 周时，面部各突起如未能正常联合，则形成面部发育异常。
![[图1-6 面部发生的部位.png|224]]

#### （一）唇裂
多见于上唇。

| 唇裂           | 成因                                               |
| ------------ | ------------------------------------------------ |
| 单侧唇裂（**较多**） | 单侧 [[球状突]] 与同侧 [[上颌突]] 未联合或部分联合所致                |
| 双侧唇裂         | ==双侧球状突==与==同侧上颌突==未联合或部分联合所致                    |
| 正中唇裂         | ==两侧球状突==之间未联合或部分联合；<br>**下唇裂**：==两侧下颌突==在中缝处未联合 |

#### （二）面裂
较唇裂少见得多。

| 面裂  | 成因                |
| --- | ----------------- |
| 横面裂 | ==上、下颌突==未联合或部分联合 |
| 斜面裂 | ==上颌突及侧鼻突==未联合    |

横面裂联合过少：小口畸形。
联合过多：大口畸形。

## 第三节 腭部的发育

### 一、腭的发育过程

腭的发育来自于**前腭突**（又叫原发腭，==来源于中鼻突==，不是来源于球状突，球状突仅参与其形成）及**侧腭突**（又叫==继发腭==，占主要部分，==来源于上颌突==）。

| 时间           | 腭部突起发育过程                                            | 位置与方向             |
| ------------ | --------------------------------------------------- | ----------------- |
| 胚胎第 6 周      | 中鼻突形成了**前腭突（原发腭）**                                  | 前颌骨和上颌切牙          |
| 胚胎第 7 周      | 上颌突自内侧向原口腔内长出**侧腭突（继发腭）**                           | ==向下==或==垂直==方向生长 |
| 胚胎第 8 周      | 互相接触                                                | ==水平==方向的转动并向中线生长 |
| 胚胎第 9 周～3 个月 | **融合**：==侧腭突与鼻中隔发生融合==                              | 从中点由前向后           |
|              | 联合：侧腭突与前腭突==向前==联合；两侧侧腭突互相==向后==联合；中心留下==切牙管==→切牙乳头 | 从中点由后向前（斜前）       |

![[图1-7 前腭突的形成、腭部突起的发育及融合.png|795]]

### 二、腭部发育异常

| 裂              | 成因                                  |
| -------------- | ----------------------------------- |
| **腭裂**（9～12 周） | ==两个侧腭突==之间及其与==鼻中隔之间==**未融合或部分融合** |
| **上颌裂**        | ==前腭突与上颌突之间==未联合或部分联合               |
| **下颌裂**        | ==两侧下颌突==未联合或部分联合                   |

在腭突的融合缝隙中，有时有上皮残留，可发生发育性囊肿，如鼻腭囊肿、正中囊肿。

## 第四节 舌的发育

### 一、舌的发育过程

|    鳃弓    |         胚胎第 4 周发育过程          |        胚胎第 6 周发育过程        |
| :------: | :--------------------------: | :-----------------------: |
|  第 1 鳃弓  |           侧舌隆突、奇结节           | 侧舌隆突越过奇结节形成舌前 2/3（**舌体**） |
|  第 2 鳃弓  |         联合突（copula）          | 鳃下隆起掩盖联合突形成舌后 1/3（**舌根**） |
| 第 3、4 鳃弓 | 鳃下隆起 (hypobranchial eminence) |      舌后 1/3（**舌根**）       |

**侧舌隆突**（lateral lingual prominence/swelling）：下颌突的原始口腔侧，内部的间充质不断增生，形成 3 个膨隆的突起。其中两个对称的隆起体积较大，称侧舌隆突。
**奇结节**（tuberculum impar）：在侧舌隆突稍下方中线处为一个小突起，称奇结节。
==界沟==（sulcus terminalis）：舌体与舌根联合线处形成的一个浅沟称界沟。

| 时间       | 甲状腺的发育                                                              |
| -------- | ------------------------------------------------------------------- |
| 胚胎第 4 周  | 奇结节沿中线向上皮深部增生，形成==甲状舌管==                                            |
| 胚胎第 6 周末 | 甲状舌管逐渐退化消失（主体消失），此时甲状腺原基已到达位置。                                      |
| 胚胎第 7 周  | 甲状舌管增生至颈部甲状软骨处，迅速发育成**甲状腺**。其发生处的舌背表面留下一浅凹即==舌盲孔==（原奇结节位置，甲状舌管起始位置） |
| 胚胎 14 周  | 味蕾发育                                                                |
| 胚胎第 3 个月 | 甲状腺有含胶质的滤泡出现，开始具有功能。                                                |

**舌体表面**被覆==外胚层上皮==，**舌根表面**被覆==内胚层上皮==，**甲状腺表面**被覆==内胚层上皮==

### 二、发育异常

| 异常     | 病因                                                                        |
| ------ | ------------------------------------------------------------------------- |
| 异位甲状腺  | ==甲状舌管上皮==在下行途中发生停滞形成异位的甲状腺                                               |
| 甲状舌管囊肿 | 甲状舌管退化后的==上皮残留==                                                          |
| 正中菱形舌  | 1.侧舌隆突==未完全掩盖奇结节==所形成的一种现象；<br>2.也可能是==局限性慢性真菌感染==（白色念球菌）。舌盲孔前方；菱形；舌乳头萎缩。 |
| 分叉舌    | ==侧舌隆突==未联合,罕见                                                            |

## 第六节 上、下颌骨的发育

### 一、下颌骨

发育自**第 1 鳃弓**。
- 第 6 周，出现下颌软骨，又叫 ==Mekel 软骨==，分别为左右第一鳃弓的条形透明软骨棒。仅起到支架作用，无功能。
- 第 7 周，细胞凝聚区分化出成骨细胞、出现膜内骨化，==形成最初的下颌骨骨化中心==。
- 第 10 周，==下颌骨体==基本形成。
- 第 12 周，==髁突软骨==出现。
- 第 4 个月，==喙突软骨==出现。

### 二、上颌骨

发育自**第 1 鳃弓**。
- ==胚胎第 8 周==，鼻囊外侧的上颌带状细胞凝聚区开始骨化。
  ① 向上形成上颌骨额突并支持眶部。
  ② 向后形成颧突。
  ③ 向内形成腭突。
  ④ 向下形成牙槽突。
  ⑤ 向前形成上颌的表面组织。
